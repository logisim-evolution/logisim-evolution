Tampon

Librairie: Gates
Introduction: 2.0 Beta 1
ANSI: IEC:

Comportement

Le tampon transmet simplement à sa sortie droite toute entrée qu'il reçoit du côté gauche. La propriété Valeur de sortie permet de modifier le comportement de la sortie selon trois modes. La table de vérité d'un tampon à un bit est la suivante.

Entrée Sorties
  O/1   O/flt. flt./1
0 0 0 U
1 1 U 1
U U U U
E E E E

Dans tous les cas si l'entrée est non spécifiée (c'est-à-dire flottante U), la sortie sera également non spécifiée. Si l'entrée est une valeur d'erreur (E), alors la sortie le sera aussi.

Les tampons sont les plus inutiles des composants de portes fournis dans Logisim; leur présence dans la bibliothèque Gates est autant une question d'exhaustivité, un composant pour chaque table de vérité à une entrée possible qu'une question de fonctionnalité utile. Néanmoins, il peut être occasionnellement utile de s'assurer que les valeurs se propagent dans une seule direction le long d'un fil.

Pins (en supposant que le composant est orienté vers l'est)

Bord ouest :
L'entrée dans le composant sa largeur de données est définie par l'attribut largeur de données.
Bord est :
La sortie, qui correspond toujours à l'entrée sur le côté gauche. Sa largeur de données est définie par l'attribut largeur de données. Le mode de présentation des données sur la sortie est défini par l'attribut Valeur de sortie.

Attributs

Lorsque le composant est sélectionné ou ajouté, Alt-0 à Alt-9 modifie l'attribut Largeur de données, et les touches fléchées modifient son attribut Orientation.

Orientation
La direction du composant (sa sortie par rapport à son entrée).
largeur de données
La largeur de bit des entrées et sorties du composant.
Valeur de sortie
Indique comment les résultats faux et vrais doivent être traduits sur la sortie. Par défaut, faux est indiqué par une tension faible (0) et vrai par une tension élevée (1), mais l'un ou l'autre peut être remplacé par une valeur à haute impédance (flottante U). Cela permet des connexions de type ou-câblé et et-câblé, comme illustré dans la documentation sur les portes AND/OR/NAND/NOR.
Étiquette
Le texte de l'étiquette associée à la porte.
Police des étiquettes
La police avec laquelle l'étiquette doit être rendue.

Comportement de l'outil pousser

Aucun.

Comportement de l'outil text

Permets de modifier l'étiquette associée à la porte.
