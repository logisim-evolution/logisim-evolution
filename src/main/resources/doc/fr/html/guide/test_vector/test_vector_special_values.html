Valeurs spéciales

Le format Vecteur de test prend en charge deux valeurs spéciales pour des tests plus flexibles :

  • <DC> (Peu importe) : Lorsqu'elle est utilisée pour une broche de sortie, cette valeur passe toujours quelle que soit la sortie réelle. C'est utile lorsque vous ne vous souciez pas de certaines sorties pendant un test. Insensible à la casse (<dc> fonctionne aussi).
  • <float> : Lorsqu'elle est utilisée pour une broche d'entrée, cela applique une valeur flottante (haute impédance). Lorsqu'elle est utilisée pour une broche de sortie, cela attend que la sortie soit flottante (valeur INCONNUE). Insensible à la casse (<FLOAT> fonctionne aussi).

Voici un exemple utilisant des valeurs spéciales :

# Test avec valeurs peu importe et flottantes
Input   Enable Output  <set> <seq>
1       1      <DC>    0     0
<float> 0      <float> 0     0

Dans le premier test, la sortie est marquée comme peu importe, donc toute valeur de sortie passera. Dans le deuxième test, l'entrée est flottante et la sortie est attendue être flottante également.
