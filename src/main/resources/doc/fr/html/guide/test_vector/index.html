Vecteurs de test

La fonctionnalité Vecteur de test vous permet de vérifier vos circuits en exécutant des tests automatisés définis dans des fichiers texte. Cette fonctionnalité prend en charge les tests de circuits combinatoires et séquentiels, ce qui la rend idéale pour vérifier des conceptions de logique numérique complexes, y compris les machines à états, les compteurs et les composants mémoire.

Le module Vecteur de test exécute une copie séparée du simulateur de circuit, il n'interfère donc pas avec la simulation dans la fenêtre du projet. Toutes les sorties incorrectes seront signalées en rouge, et vous pouvez survoler la boîte rouge avec la souris pour voir ce que la sortie aurait dû être selon le vecteur de test.
