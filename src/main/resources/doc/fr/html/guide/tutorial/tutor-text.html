Etape 3: Ajouter du texte

Le text n'est pas indispensable aux fonctionnements des circuits , mais si vous voulez présenter votre circuit à quelqu'un (comme un enseignant), alors quelques étiquettes permettront de mieux présenter le rôle des différents éléments de votre circuit.

Sélectionnez l'outil "Texte" (). Vous pouvez cliquer sur une broche d'entrée et commencer à taper pour lui assigner un label. (Il est préférable de cliquer directement sur le symbole de l'entrée plutôt que de cliquer à l'endroit où vous voulez placer le texte, car de cette manière l'étiquette sera liée avec le symbole). Vous pouvez faire de même avec la broche de sortie. Vous pouvez aussi cliquer ou bon vous semble pour placer du text explicatif non attaché a un symbole du schéma.

De nombreux composants acceptent des labels, par exemple si vous cliquez dans une des portes vous pouvez aussi lui assigner un label

Il est possible de modifier un label de plusieurs autres manières.

    Avec un double-clic sur le composant avec l'outil Edition().
    Avec un clic sur un label avec l'outil "Text"().
    En modifiant la propriété "label"dans la table des attributs.
