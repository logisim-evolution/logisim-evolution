Fenêtre chronogramme

Pour comprendre ou débugger un circuit, il est souvent très utile de pouvoir observer les différents signaux de manière visuels, c'est le but du chronogramme. Ce module permet l'enregistrement des signaux sous forme graphique ou dans un tableau de valeur dans un fichier text.

Vous pouvez entrer dans le module de journalisation via le menu | Simulation |→| Chronogramme |. Il fait apparaître la fenêtre de sélection des signaux.

Le circuit ci-dessous est un exemple d'illustration pour le module chronogramme.

Il y a deux signaux nommés qui servent de pilote à l'affichage des chronogrammes. Un est impératif et il est nommé sysclk et l'autre une horloge optionnelle nommée clk. sysclk est très souvent une horloge, mais pas nécessairement, cela peut aussi être une entrée ou une sortie. L'horloge clk en association avec une horloge sysclk permet d'enregistrer un chronogramme sur une période déterminée. Selon le paramètre de simulation "Duration of main tick" l'enregistrement sera arrêté en fonction de l'évolution de clk.

Note: il est impératif qu'un signal nommé sysclk apparaisse dans votre circuit. Il sera utilisé comme base de temps par le module chronogramme. Il n'est pas nécessaire qu'il soit connecté à votre circuit. Si c'est une horloge, elle est en principe la plus rapide.
