Sélection des signaux

Pour comprendre ou débugger un circuit, il est souvent très utile de pouvoir observer les différents signaux de manière visuels, c'est le but du chronogramme. Ce module permet la présentation et l'enregistrement des signaux sous forme graphique.

Vous pouvez entrer dans le module de journalisation via le menu | Simulation |→| Chronogramme |. Il fait apparaître la fenêtre Chronogramme.
