Le menu Editer

Annuler XX  Ctrl-Z

Annule l'action la plus récente (XX) sur le dessin du circuit comme il serait enregistré dans un fichier. Notez que cela n'inclut pas les modifications de l'état de simulation du circuit (comme avec les manipulations effectuées par l'outil: pousser).

Rétablir XX  Ctrl-R

Permets de revenir sur une annulation du menu précédent.

Couper  Ctrl-X

Supprime du dessin les composants du circuit actuellement sélectionnés.

Note: Le presse-papiers de Logisim est géré séparément du presse-papiers de l'ensemble du système. En conséquence, couper / copier / coller ne fonctionnera pas dans différentes applications, même en incluant d'autres copies courantes de Logisim. Si, toutefois, plusieurs projets sont ouverts sous le même processus Logisim, vous devriez pouvoir couper / copier / coller entre eux.

Copier  Ctrl-C

Copie les composants actuellement sélectionnés dans le circuit dans le presse-papiers Logisim. (Voir la remarque sous l'élément de menu Couper.)

Coller  Ctrl-V

Colle les composants du presse-papiers de Logisim dans la sélection en cours. (Voir la remarque sous l'élément de menu Couper).

Lorsque vous collez des composants, ils seront dessinés en gris clair. Ils ne seront pas réellement «déposés» dans le circuit tant que vous n’avez pas déplacé la sélection, modifié la sélection ou effectué un autre clic.

La raison de ce comportement est la suivante: pour être cohérent avec son autre comportement, Logisim doit immédiatement fusionner tous les fils dès qu’ils sont déposés dans un circuit. Ce processus de fusion modifie les fils existants dans le circuit. Lorsque vous collez des fils du presse-papiers, vous souhaiterez probablement qu'ils apparaissent à un autre endroit, et le changement inhérent au processus de fusion serait contraire à vos souhaits.

Supprimer  Del / Delete

Supprime tous les composants de la sélection en cours du circuit, sans modifier le presse-papiers.

Dupliquer  Ctrl-D

Crée une copie de tous les composants de la sélection en cours. Cela revient à sélectionner Copier, puis Coller, sauf que Dupliquer ne modifie ni n’utilise le Presse-papiers.

Tout sélectionner  Ctrl-A

Sélectionne tous les composants du circuit actuel.

Élever la sélection  Ctrl-Haut

Cet élément de menu est disponible uniquement lors de l'édition de l'apparence d'un circuit. Il soulève le ou les objets actuellement sélectionnés de manière à ce qu'ils soient dessinés au-dessus d'un objet que la sélection chevauche. Si la sélection est recouverte par plusieurs objets, elle est remontée pour être au-dessus de l'élément le plus bas non sélectionné. Répétez l'opération à plusieurs reprises jusqu'à obtenir l'ordre désiré.

(Il est difficile de déterminer si deux objets arbitraires se chevauchent. Logisim utilise un algorithme de sélection de plusieurs points aléatoires dans chacun des deux objets et vérifie si un point se trouve également dans l’autre objet. Parfois, le chevauchement est faible, disons moins de 5% de l'un des objets.)

Abaisser la sélection  Ctrl-Bas

Cet élément de menu est disponible uniquement lors de l'édition de l'apparence d'un circuit. Il abaisse le ou les objets actuellement sélectionnés de manière à ce qu’ils soient dessinés au-dessous d’un objet que la sélection chevauche actuellement. Si la sélection chevauche plusieurs objets, elle est abaissée pour être inférieure à l'objet le plus haut non sélectionné. Répétez l'opération à plusieurs reprises jusqu'à obtenir l'ordre désiré.

Élever au premier plan  Ctrl-Maj-Haut

Disponible uniquement dans le mode "modification de l'apparence" d'un circuit, cet élément de menu force le ou les objets actuellement sélectionnés à se dessiner par-dessus tous les autres objets. (L’ancre et les ports sont des exceptions; ils sont toujours au dessus.)

Abaisser à l'arrière-plan  Ctrl-Maj-Bas

Disponible uniquement lors de la modification de l'apparence d'un circuit, cet élément de menu force le ou les objets actuellement sélectionnés à se dessiner en dessous tous les autres objets.

Ajouter un sommet

Disponible uniquement lors de la modification de l'apparence d'un circuit et nécessite qu'un point soit sélectionné sur une ligne, une polyligne ou un polygone, cet élément de menu insère un nouveau sommet sur la forme. Avant l'insertion, le point sélectionné est dessiné comme un diamant.

Supprimer un sommet

Disponible uniquement lorsque vous modifiez l'apparence d'un circuit et qu'un sommet existant a été sélectionné sur une polyligne ou un polygone, cet élément de menu supprime le sommet sélectionné. Avant la suppression, le sommet sélectionné est dessiné comme un losange dans le carré représentant le sommet. Logisim ne permettra pas de supprimer un sommet sur un polygone avec seulement trois sommets ou sur une polyligne avec seulement deux sommets.
