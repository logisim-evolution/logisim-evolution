Valores Especiais

O formato de Vetor de Teste suporta dois valores especiais para testes mais flexíveis:

  • <DC> (Não Importa): Quando usado para um pino de saída, este valor sempre passa independentemente da saída real. Isso é útil quando você não se importa com certas saídas durante um teste. Não diferencia maiúsculas/minúsculas (<dc> também funciona).
  • <float>: Quando usado para um pino de entrada, isso aciona um valor flutuante (alta impedância). Quando usado para um pino de saída, isso espera que a saída seja flutuante (valor DESCONHECIDO). Não diferencia maiúsculas/minúsculas (<FLOAT> também funciona).

Aqui está um exemplo usando valores especiais:

# Teste com valores "não importa" e flutuantes
Input   Enable Output  <set> <seq>
1       1      <DC>    0     0
<float> 0      <float> 0     0

No primeiro teste, a saída está marcada como "não importa", então qualquer valor de saída passará. No segundo teste, a entrada está flutuante e a saída é esperada ser flutuante também.
