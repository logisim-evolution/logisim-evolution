Colocação de componentes em uma matriz.

Em Logisim-evolução é possível colocar componentes matriciais no seu esquema. Para o fazer, proceda da seguinte forma:
Selecione a ferramenta desejada no painel de navegação, por exemplo, o LED, e mova-a para a superfície de trabalho e antes de clicar para soltá-la pressione a tecla X . A sombra do componente irá transformar-se numa pequena matriz 2 por 2 do componente.

Clique na parte superior esquerda da posição onde você quer colocar a futura matriz. E preencha os dados do formulário. Por exemplo: 4x5 elementos com um espaço de 4.

Se a auto-etiquetagem estiver ativada, os componentes serão etiquetados como _X_Y, Exemplo: Led_X1_Y0
