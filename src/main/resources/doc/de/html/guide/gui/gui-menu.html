Menüs

In diesem Abschnitt werden schnell die sieben Menüs erklärt, die jedes wichtige Logisim-Fenster begleiten.

Das Menü Datei : Dieses Menü ermöglicht das Öffnen und Schließen, um Projekte zu speichern. Sie ermöglicht den Zugriff auf die préférences und das Beenden des Programms.

Das Menü Bearbeiten : Wird hauptsächlich zum Bearbeiten, Kopieren, Löschen, Duplizieren, Verschieben von Zeichnungselementen verwendet.

Das Menü Projekt : Ermöglicht die Verwaltung von Bibliotheken, die Änderung der Reihenfolge im Navigationsbereich und die Änderung des Erscheinungsbildes dieses Bereichs. Es ermöglicht den Zugriff auf die Projekteinstellungen.

Das Menü Simulation : Steuerung der Simulation, Start, Stopp, Übergang in den Schrittmodus, Start der Uhren.

Das FPGA-Menü : Bearbeiten Sie das Bild einer FPGA-Platine, verwalten Sie die Synthese und das Laden des Schaltplans in die Platine.

Das Menü Fenster : Minimieren, Erweitern, Schließen von Fenstern. Gehen Sie zum Fenster Kombinationsanalyse (falls aktiviert) und zu den Anwendungseinstellungen.

Das Menü Hilfe :Gehen Sie zum Tutorial, zum Referenzhandbuch oder zur Referenz der Bibliotheken.

Sie finden vollständige Informationen über die Menüs auf den Seiten, die diesem Thema gewidmet sind.
