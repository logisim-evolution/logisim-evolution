Graphic chart

Malheureusement javahelp V2.0 est assez pauvre dans la présentation graphique et il ne supporte que le CSS V1.x. Ce document présente la charte et les constructions utilisées dans la documentation de logisim.

L'en-tête et fichier html

L'entête est assez commune, les lignes a adapter sont :
La langue de la page :META http-equiv="Content-Language" content="en"
la référence au fichier CSS selon hyérachie des dossiers :LINK rel="stylesheet" type="text/css" href="..\style.css" 


<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0//EN">
<html>
<head>
 <META http-equiv="content-type" content="text/html; charset=UTF-8">
 <META http-equiv="Content-Language" content="en">
 <title>Graphic chart</title>
<LINK rel="stylesheet" type="text/css" href="..\style.css">
</head>
<body>
<div class="maindiv">
...
...the content of the page here
...
</div>
</body>
</html>
</code>

Les titres et sous titres

Les balises <h1> <h2> <h3> sont utilisées et vous pouvez observer les titre de cette page pour les effets.
