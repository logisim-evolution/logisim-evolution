行为

寄存器存储单个多位值，该值在其矩形内以十六进制显示，并在其 Q 输出上发出。 当时钟输入（由南边的三角形表示）指示如此时，存储在寄存器中的值将更改为当时 D 输入的值。 时钟输入指示何时发生这种情况是通过触发属性配置的。

reset 输入将寄存器的值异步重置为 0（全零）； 也就是说，只要 reset 为 1，无论时钟输入如何，该值都固定为 0。

引脚

东边，标记为 Q （输出，位宽与数据位属性匹配）
输出寄存器当前存储的值。
西边，标记为 D （输入，位宽与数据位属性匹配）
数据输入：在时钟值从0上升到1的瞬间，寄存器的值改变为该时刻 D 输入的值。
西边，标记为 en （输入，位宽 1）
启用：当该值为 0 时，时钟触发被忽略。 当前值继续出现在输出上。 当该输入为 1 或未定义时，时钟触发器被启用。
南边，用三角形表示（输入，位宽1）
时钟输入：在该输入值从0上升到1（上升沿）的瞬间，寄存器的值将被更新为 D 输入的值。
南边缘，标记为 0 （输入，位宽 1）
异步复位：当为0或未定义时，该输入无效。 只要它为 1，寄存器的值就固定为 0。这是异步发生的，即不考虑当前时钟输入值。 只要该值为 1，其他输入就没有影响。

属性

当选择或添加组件时，Alt-0 到 Alt-9 会更改其 数据位 属性。

数据位
寄存器中存储的值的位宽。
扳机
配置如何解释时钟输入。 上升沿 值表示寄存器应在时钟从0上升到1的瞬间更新其值。 下降沿 值表示应在时钟从0上升到1的瞬间更新其值 时钟从1下降到0。 高电平 值表示每当时钟输入为1时寄存器应不断更新。 低电平 值表示应更新 当时钟输入为0时连续。
标签
与寄存器关联的标签内的文本。
标签字体
用于呈现标签的字体。

Poke 工具行为

单击寄存器会将键盘焦点带到寄存器（由红色矩形表示），并且键入十六进制数字将更改存储在寄存器中的值。

文本工具行为

允许编辑与组件关联的标签。
