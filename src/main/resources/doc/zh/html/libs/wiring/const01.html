库: 线路
介绍: 2.7.0
外观符号:

Behavior

将单个逻辑值发送到线路上。 对于 power 元素，( ）由三角形 表示，该值将为 1 （或者，如果数据位属性大于 1，则为全 1 的值）。 对于地面元素，( ）由三个长度递减的水平线箭头指示，该值将为 0 （或者，如果数据位属性大于 1，则为全 0 的值） 。

使用更通用的 Constant 组件可以实现相同的功能。 更喜欢接地和电源的唯一原因是它们是常用的电子符号。

引脚

每个元件只有一个引脚，即位宽与数据位属性匹配的输出。 该元件在此引脚上不断输出相同的值：对于接地元件，输出为全 0 值，对于功率组件，输出为全 1 值。

属性

选择或添加元件时， Alt-0 到 Alt-9 会更改其 数据位 属性。 快速使用按键可以输入更大的值，例如 Alt-0 ALT-1 ALT-2 给出12。箭头键改变其 方向 属性。

方向
元件箭头 点 的方向，与其引脚位置相反。
数据位
放置到线路上的值的位宽度。
