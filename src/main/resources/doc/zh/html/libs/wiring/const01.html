电源 / 接地

所属库： 线路
引入版本： 2.7.0
外观：

行为

向线路输出一个固定的逻辑值。 对于电源元件（，以三角形表示）， 该值为 1 （如果 数据位宽 属性大于 1，则输出全 1 的值）。 对于接地元件（， 以三条逐渐变短的水平线箭头表示）， 该值为 0 （如果 数据位宽 属性大于 1，则输出全 0 的值）。

使用功能更灵活的 常量 组件也可以实现相同的功能。 选择电源和接地组件的唯一原因是它们是电子电路中常见且直观的符号。

引脚

每个组件只有一个引脚，这是一个输出引脚，其位宽与 数据位宽 属性一致。 该组件会在此引脚上持续输出同一个值： 对于接地组件，输出始终为全 0； 对于电源组件，输出始终为全 1。

属性

当组件被选中或正在放置时， 可以使用 Alt-0 到 Alt-9 来修改其 数据位宽 属性。 快速连续按键可以输入更大的数值， 例如依次按下 Alt-0、Alt-1、Alt-2 将得到数值 12。 方向键用于修改其 朝向 属性。

朝向
组件箭头“尖端”的方向，该方向与其引脚所在的位置相反。
数据位宽
输出到线路上的数值的位宽。
