Clock

库: 线路
介绍: 2.0 Beta 13 (in Base library, moved to Wiring in 2.7.0)
外观符号:

行为

只要通过 模拟菜单 启用时钟，时钟就会定期其输出值 。 （默认情况下禁用时钟。）“时钟”是 Logisim 的时间单位； 可以从 模拟 菜单的子菜单 时钟滴答频率 中选择滴答发生的速度。

可以使用其 高持续时间 和 低持续时间属性 来配置时钟周期。

在 分步模式 每次点击时钟都会前进一格。

请注意 Logisim-evolution 对时钟的模拟是理想状态的：在实际电路中，多个时钟会相互漂移，并且永远不会同步移动。 但在 Logisim-evolution 中，所有时钟都以相同的速率经历滴答声。

引脚

时钟只有一个引脚，一个位宽为1的输出，其值将代表时钟的当前值。 此引脚的位置在 方向 属性中指定。
每当启用时钟时，时钟的值都会按其时间表切换，并且每次都会前进一个刻度。

属性

当选择或添加组件时，箭头键会更改其 方向 属性。

方向
元件输出引脚所在的一侧。
高持续时间
每个周期内时钟输出应为 1 的时间长度。
持续时间短
每个周期内时钟输出应为 0 的时间长度。
相位偏移
时钟滴答声中的相位偏移。 下面是两个相同的时钟，其中一个（灰色）的相移为 1 个刻度。
标签
与时钟组件关联的标签内的文本。
标签位置
标签相对于元件的位置。
标签字体
用于呈现标签的字体。

Poke 工具行为

通过单击时钟元件，您可以将所有时钟前进一格。

文本工具行为

允许编辑与元件关联的标签。
