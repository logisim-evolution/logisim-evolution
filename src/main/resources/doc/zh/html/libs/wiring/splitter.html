分线器

库： 线路
引入版本： 2.0 Beta 1（在基础库中引入，2.7.0 中移至线路库）
外观：

行为

分线器用于在多位信号与其若干位子集之间建立对应关系。它既可以将一个多位信号拆分为若干部分，也可以将多个部分合并为一个多位信号，甚至可以同时完成这两种操作。关于分线器的更完整说明，请参见 分线器 章节。

分线器以及导线没有传播延迟。

注意：分线器（splitter） 这一术语并非标准电子学术语，是 Logisim 特有的。

引脚

为了区分分线器的多个连接点，我们将一侧的单个连接点称为 合并端，另一侧的多个连接点称为 分支端（见下图）。

合并端 分支端
合并端：
合并端的位宽由 输入位宽 属性决定，表示通过该总线的所有导线数量。
分支端：
每个分支端的位宽通过属性位 x 确定。

属性

当组件被选中或正在放置时，数字键 0–9 可修改 分支数， Alt-0–Alt-9 可同时修改 分支数 和 输入位宽， 方向键用于修改 朝向。

朝向
分支端相对于合并端的位置方向。
分支数
分支端的数量。
输入位宽
合并端的位宽。
外观
用于指定分线器在电路中的绘制方式。 左手形（默认）表示分支端从合并端向左延伸； 右手形 表示分支端从合并端向右延伸； 居中 表示分支端向左右大致等距延伸； 传统 表示为每个分线端绘制不带标签的延伸线，主要用于兼容 2.7.0 之前的版本。
左手形 右手形 居中 传统
间距
分支端之间的间距。
位 x
指定合并端中的第 x 位对应到哪个分支端。 分支端的编号从 0 开始：当朝向为东西时自上而下编号，当朝向为南北时自左向右编号。 某一位可以不对应任何分支端，但不能同时对应多个分支端。
