上/下拉电阻

库: 线路
介绍: 2.5.0 (in Base library, moved to Wiring in 2.7.0)
外观符号:
ANSI: IEC:

行为

当连接到一个点时，仅当该点的值为动态值（ U ）时，该元件才起作用。 在这种情况下，电阻器会将与其连接的电线拉向其 拉力方向 属性中指示的值。

如果它连接到多位值，则动态值中的每个位都会按指定的方向拉动，而不浮动的位则保持不变。

引脚

该电阻器只有一个引脚，该引脚是输出端，其位宽源自其所连接的任何组件。

属性

当选择或添加元件时，箭头键会更改其 方向 属性。

方向
元件引脚距元件中心的方向。
拉动方向
指定应将动态值拉至的值。 这可能是 0 、 1 或错误值 ( E ) 。
