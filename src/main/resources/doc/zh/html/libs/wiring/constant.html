常量

库: 线路
介绍: 2.0 Beta 1 (in Gates library, moved to Wiring in 2.7.0)
外观符号:

行为

发出其 Value 属性中指定的值。

引脚

只有一个引脚，即位宽与数据位属性匹配的输出。 该引脚的位置在 方向 属性中指定。 该元件不断地在此引脚上输出 Value 属性中指定的任何值。

属性

选择或添加元件时，十六进制数字 0 到 9 和 a< /b> 通过 f 更改其 Value 属性。 Alt-0 到 Alt-9 更改其 数据位 属性和 箭头键更改其 方向 属性。

方向
引脚相对于绘制值的位置的方向。
数据位
放置到线路上的值的位宽度。
值
由元件发出的以十六进制编写的值。 用于指定值的位数不能超过组件的位宽度。
