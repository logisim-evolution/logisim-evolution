探针

库: 线路
介绍: 2.0.3 (in Base library, moved to Wiring in 2.7.0)
外观符号:
箭头形状 经典形状

行为

探针是一种简单地显示电路中给定点的值的元件。 它本身不与其他元件交互。

在大多数方面，探针元件复制了配置为输出引脚的引脚元件中的功能。 主要区别在于，如果电路用作子电路元件，则输出引脚将成为该接口的一部分，而探针则不是。 它们的不同之处还在于探针没有要配置的数据位属性：位宽度是根据它在输入上碰巧看到的任何值推断出来的。 从图形上看，它们相似，但边框略有不同：引脚具有粗的黑色边框，而探针具有细的灰色边框。

引脚

探针元件只有一个引脚，用作探针的输入。 该引脚接受的宽度是自适应的：探针将适应任何宽度的输入。

属性

当选择或添加元件时，箭头键会更改其 方向 属性。

方向
元件输入引脚所在的一侧。
基数
允许您定义表示值的基数。
二进制 1010 的示例
 Binary : 1010b
 Octal : 12o
 Signed decimal : -6s
 Unsigned deciaml : 10u
 Hexadecimal : Ah
 Float : 1.4E-44f 仅在 32 或 64 位中有值 NaNf
标签
与元件关联的标签内的文本,注意标签的文本：必须以字母开头，只能包含字符[a-Za-Z_0-9]，不能包含多个连接的“_”，不能以“_”结尾，不能是保留的 VHDL 关键字，不能是保留的 Verilog 关键字。
标签位置
标签位置与元件相关
标签字体
用于设置标签的字体。
