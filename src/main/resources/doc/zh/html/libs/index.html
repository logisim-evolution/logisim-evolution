库参考

Logisim 库由一组工具组成，可通过在画布区域单击和拖动鼠标来与电路交互。通常，工具用于将特定类型的组件添加到电路中；其中一些常用工具，例如手形工具（）和选择工具（），还可用于查看、选择或调整已有组件。

通常，当组件被选中或添加时，使用Alt-0至Alt-9来更改其数据位宽属性。快速使用按键可以输入更大的值，例如：ALT-1 ALT-2得到 12。

Logisim 内置库中包含的所有工具均在此参考资料中进行了说明。
