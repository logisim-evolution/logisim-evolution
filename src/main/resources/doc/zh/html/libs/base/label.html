标签

库： 基础
引入版本： 2.0 Beta 1
外观：

行为

这是一个简单的文本标签，可以放置在电路中的任何位置。它不会以任何方式与电路中传输的值进行交互，只是在绘制电路时可见。

与当前内置库中的所有其他组件不同，标签组件可以放置在画布上的任何位置；它们不会吸附到网格。

引脚

无。

属性

文本
标签中显示的文本。此值可以在属性表中编辑，也可以使用文本工具在画布上编辑。
字体
绘制标签时使用的字体。
水平对齐
文本相对于标签官方位置（创建标签时鼠标点击的位置）的水平定位方式。“左”表示文本应绘制为使其左边缘位于该位置；“右”表示文本应绘制为使其右边缘位于该位置；“居中”表示文本应绘制为使其水平中心位于该位置。
垂直对齐

文本相对于标签官方位置（创建标签时鼠标点击的位置）的垂直定位方式。“基线”表示基线应与该位置相交；“顶部”表示文本的顶部应与该位置相交；“底部”表示文本的底部应与该位置相交；“居中”表示文本应在该位置垂直居中。

文本的顶部和底部是根据字体的标准上升和下降值计算的；因此，即使实际文本中不包含高字母（如b）或下降字母（如g），在进行垂直定位时也会假定包含此类字母。

手形工具行为

无。

文本工具行为

允许编辑标签中显示的文本。
