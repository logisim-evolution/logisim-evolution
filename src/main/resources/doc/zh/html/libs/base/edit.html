编辑工具

库： 基础
引入版本： 2.3.0

行为

编辑工具允许用户重新排列现有组件并添加导线。该工具的具体行为取决于用户在画布上按下鼠标的位置。

  • 当鼠标悬停在现有组件的引脚上，或位于当前导线上方时，编辑工具会在鼠标位置周围显示一个绿色小圆圈。在该处按下按钮将开始添加新导线。但如果用户在释放按钮前未将鼠标拖动足够远以启动导线，则该按下操作将被视为鼠标单击，因此该导线仅会被添加到当前选择中。

    所添加导线的位宽根据其所连接的组件推断得出。如果导线未连接到任何组件，则导线将显示为灰色，表示其位宽未知；如果导线所连接的各组件在位宽上不一致，则导线将显示为橙色以指示冲突，并且该导线实际上将拒绝传输任何值，直到用户解决该冲突为止。

    Logisim 中的所有导线均为水平或垂直走向，绝无对角线。

    导线是无方向的；也就是说，它们可以从任一端点向另一端传输值。实际上，导线可以同时双向传输值：在下面的示例中，一个比特从左侧的上方输入流经中央导线，然后通过中央导线回流，接着再次通过中央导线向前流动，最终到达右下方的输出。

    一次鼠标拖动可以创建多个导线线段。其精确过程在描述上有些令人困惑，但在实践中却非常直观：如果您使用接线工具请求一条特定的导线线段，该线段将在遇到现有组件的引脚或现有导线线段的端点处被分割。此外，如果任何新导线线段的端点恰好落在现有导线的中间某处，那么该现有导线自身也将被分割成多个线段。

    您也可以通过在线段的端点处开始拖动，然后沿该线段向后绘制，来缩短或删除现有的导线线段。在拖动过程中，缩短操作通过在被移除的导线部分上绘制一条白线来指示。

    某些组件会绘制出可供导线连接的短截线，例如或门和受控缓冲器。Logisim 会静默地纠正那些试图创建略微超出截线末端的导线的操作。

  • 然而，如果用户在导线中间某点按下 Alt 键，绿色圆圈将消失。此时按下鼠标将选中该导线，拖动鼠标则会移动它。

  • 在当前已选中的组件内部按下鼠标按钮，将开始拖动并移动选择中的所有元素。

    默认情况下，Logisim 会计算添加新导线的方式，以确保在移动过程中不会丢失任何现有连接。（有时它会删除或缩短现有导线。）如果您在执行移动时不希望进行这些更改，可以在移动过程中按下 Shift 键。如果您想完全禁用此行为，请转到项目 > 选项，选择画布选项卡，并取消选中移动时保持连接框；在这种情况下，仅当按下 Shift 键时才会计算连接。

    拖动选择可能导致导线出现意外行为：如果您拖动一个包含某些导线的选择到其他一些导线上，所有导线将被合并，并且合并后的导线将被放入选择中。因此，如果您第二次拖动该选择，原先位于该位置的导线将不会被留下。这种行为对于保持 Logisim 中导线从不重叠的直观行为是必要的。并且这通常不会构成重大问题：Logisim 会在放置过程中绘制完整的选择，您应该在确认其位于正确位置后再放置。

  • 在未选中的组件内部（但不在该组件的接线点上）按下鼠标，将取消当前选择中的所有组件，并改为选择包含单击位置的一个或多个组件。

  • 按住 Shift 键并在组件内部单击鼠标，将切换该组件是否属于当前选择。如果多个组件包含同一位置，则所有这些组件的选择状态都会被切换。

  • 从不在任何组件内的位置开始拖动鼠标，将取消当前选择中的所有组件，并启动矩形选择。矩形所包含的所有组件将被放入选择中。

  • 按住 Shift 键并从不在任何组件内的位置开始拖动鼠标，将启动矩形选择。矩形所包含的所有组件的选择状态都会被切换。

  • 但是，如果在不在任何组件内的位置按下 Alt 键，这将启动新导线的添加。在这种情况下会绘制一个绿色小圆圈来指示。

在选择中选定所需项目后，您当然可以通过编辑菜单对所有项目进行剪切/复制/粘贴/删除/创建副本操作。

某些按键在使用编辑工具时有效。

  • 方向键会更改选中所有具有朝向属性的组件的朝向属性。

  • Delete 和 Backspace 键会从电路中删除选中的所有内容。

  • Insert 和 MenuKey-D 键会创建当前选中组件的副本。

Logisim 在复制选中内容或将剪贴板粘贴到电路中时的行为有些特殊：它不会立即将组件放入电路；相反，选中内容将是一组“幽灵”，一旦它们被拖到其他位置或从选中内容中移除，就会被放入电路。（这种特殊行为是必要的，因为粘贴操作否则会立即将选中内容的导线合并到当前电路中，并且如果用户想要将粘贴的组件移动到其他地方，之前存在的导线会随粘贴的剪贴板一起被拖动。）

属性

无。不过，选中一个组件会显示其属性。选中多个组件时，会显示所有组件共享的属性，如果属性值不同则显示为空白，否则显示它们共有的值。（如果选中内容中有非导线组件，则忽略导线。）对属性值的更改会影响所有选中的组件。
