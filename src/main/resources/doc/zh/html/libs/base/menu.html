菜单工具

库： 基础
引入版本： 2.0 Beta 1

行为

菜单工具用于打开现有组件的弹出菜单。默认情况下，右键单击或按住 Control 键并单击组件会弹出此菜单；也可以在项目选项的鼠标选项卡中配置鼠标按钮的行为。

大多数组件的弹出菜单包含两个项目。

  • 删除：从电路中移除该组件。
  • 显示属性：将组件的属性放入窗口的属性表中，以便查看和更改属性值。

然而，对于某些组件，菜单包含额外的项目。子电路（即在另一个电路中将一个电路用作“黑盒”的实例）就是一个例子：除了上述两个项目外，弹出菜单还包括另一个项目。

  • 查看 XXX：将正在查看和编辑的电路布局更改为子电路的布局。布局中看到的值将与上层电路处于同一层次结构中。（参见用户指南中的“调试子电路”一节。）

其他组件也可能扩展弹出菜单。在当前版本的 Logisim 内置库中，唯一这样的组件是RAM和ROM。

属性

无。
