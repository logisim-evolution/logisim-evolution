文本工具

库： 基础
引入版本： 2.0 Beta 1

行为

文本工具允许您创建和编辑与组件关联的标签。哪些组件支持标签在其文档的“文本工具行为”部分中说明。在当前版本中，内置库中的以下组件支持标签。

基础库 引脚
时钟
标签
探针

寄存器
计数器
移位寄存器
随机数发生器
输入/输出库 按钮
LED

对于可以添加标签但当前未分配标签的组件，您可以单击组件内的任意位置来添加标签。如果已有标签，则需要单击标签内部。如果您单击的位置当前没有可编辑的标签，Logisim 将开始添加一个新的标签组件。

在当前版本的 Logisim 中，文本编辑功能仍然相当基础。无法在标签内选择文本区域。也无法在标签中插入换行符。

属性

该工具的属性与标签组件相同。在编辑现有组件的标签时，这些属性不会产生影响，但它们会赋予使用文本工具创建的任何标签。

单击支持文本工具的组件将显示该组件的属性。
