接线工具

库： 基础
引入版本： 2.0 Beta 1

行为

接线工具用于创建导线线段，将值从一个端点传输到另一个端点。这些值的位宽可以是任意值；具体位宽会根据导线最终连接的组件自动推断。如果导线未连接到任何组件，则会以灰色绘制，表示其位宽未知；如果导线所连接位置的组件在位宽上不一致，则导线会以橙色绘制，表示存在冲突，并且该导线实际上会拒绝传输任何值，直到用户解决冲突为止。

一次鼠标拖动可以创建多个导线线段。这个过程文字描述起来略显复杂，但实际操作很直观：如果使用接线工具请求绘制一条特定的导线线段，该线段会在碰到现有组件的引脚或现有导线线段的端点处被分割。此外，如果任何新导线线段的端点落在现有导线的中间某处，那么该导线也会被分割成多个线段。

对于某些绘制了短引线以供导线连接的组件（例如或门或受控缓冲器），Logisim 会自动纠正试图创建略微超出引线末端的导线的操作。

也可以使用接线工具缩短现有的导线线段，方法是从线段的端点开始或结束拖动，并使其与现有线段重叠。

Logisim 中的所有导线要么是水平的，要么是垂直的。

导线也是无方向的；也就是说，它们可以从任一端点向另一端传输值。实际上，导线可以同时双向传输值；下面示例中的中间导线正是如此。

属性

接线工具本身没有属性，但它创建的导线具有属性。

方向
指示导线是水平还是垂直。此属性的值无法更改。
长度
指示导线有多少像素长。此属性的值无法更改。

手形工具行为

使用手形工具单击现有导线线段时，Logisim 会显示流经该导线的当前值。此行为对于多位导线特别有用，因为黑色导线本身无法显示其承载的具体值。

对于多位值，可以使用 Logisim 首选项对话框的布局选项卡来精确配置值的显示方式（例如，以二进制、十进制或十六进制显示）。
