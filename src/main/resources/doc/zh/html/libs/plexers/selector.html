位选择器

库： 复用器
引入版本： 2.0.5
外观：

行为

对一组多位输入信号，从最低位开始按等宽分组，并输出由选择输入指定的一组。

例如，8 位输入为 01010101，若输出为 3 位，则第 0 组为最低 3 位 101，第 1 组为接下来的 3 位 010，第 2 组为再往上的 3 位 001。（超过最高位的部分用 0 补齐。） 选择输入为 2 位，用于选择输出哪一组；若选择输入为 3，则输出为 000。

引脚（假设组件朝东）

西侧（输入，位宽与“数据位宽”一致）
数据输入，用于从中选取位组输出。
东侧（输出，位宽与“输出位宽”一致）
由选择输入选出的位组。
南侧（输入，位宽为“数据位宽/输出位宽”的商，向上取整）
选择输入：决定将哪一组位送到输出端。

属性

当组件被选中或正在放置时，数字键 0–9 可修改 输出位宽， Alt-0–Alt-9 可修改 数据位宽， 方向键可修改 朝向。

朝向
组件方向（其输出相对于输入的位置）。
数据位宽
数据输入的位宽。
输出位宽
输出端的位宽。

手形工具行为

无。

文本工具行为

无。
