位选择器

库: 复用器
介绍: 2.0.5
外观符号:

行为

给定多个位的输入，这会将其分为几个大小相等的组（从最低位开始）并输出由选择输入选择的组。

例如，如果我们有一个八位输入 01010101，并且我们将有一个三位输出，那么组 0 将是最低位三位 101，组 1 将是接下来的三位 010，组 2 将是接下来的三位 001。（超出顶部的任何位都用 0 填充。）选择输入将是一个两位数，用于选择要输出这三组中的哪一组； 如果选择输入为 3，则输出为 000。

引脚（假设元件朝东）

西边（输入，位宽与数据位属性匹配）
应从中选择哪些位用于输出的数据值。
东边（输出，位宽与输出位属性匹配）
数据值中的一组位，由选择输入选择。
南沿（输入，位宽是数据位和输出位的商，向上舍入）
选择输入：确定应将哪些位组路由到输出。

属性

当选择或添加元件时，数字“0”到“9”会更改其 输出位 属性，Alt-0到Alt-9会更改其 数据位 属性 ，方向键更改其 Facing 属性。

方向
元件的方向（其输出相对于其输入）。
数据位
元件数据输入的位宽度。
输出位
元件输出的位宽。
