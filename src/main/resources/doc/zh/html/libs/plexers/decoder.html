解码器

库: 复合器
介绍: 2.0 Beta 11
外观符号:

行为

仅在一个输出上发出 1； 哪个输出为 1 取决于通过南边缘输入接收到的当前值。

引脚（假设元件朝东，选择底部/左侧）

东边，可变数量（输出，位宽 1）
输出的编号从北边的 0 开始。 如果每个输出的数字与当前通过南边的选择输入接收到的值匹配，则每个输出将为 1； 否则，其值将为零或浮动，具体取决于三态的值？ 属性。 如果选择输入包含任何未指定的位，则所有输出都是浮动的。
南边缘，左侧（输入，位宽1）
启用：为 0 时，所有输出均包含所有浮动位（或零），无论选择输入如何。
南边，灰色圆圈表示的右侧（输入，位宽与 Select Bits 属性匹配）
选择输入：该输入的值决定哪个输出为 1。

属性

当选择或添加元件时，数字“1”到“4”会更改其 Select Bits 属性，并且箭头键会更改其 方向 属性。

方向
元件的方向（指定哪一侧有输出）。
选择位置
选择线和启用线相对于元件的位置。
选择位
元件南边缘的选择输入的位宽度。 解码器的输出数量将为 2 selectBits 。
三态？
指定未选择的输出应该浮动（是）还是零（否）。
禁用输出
指定当元件被禁用时（即，当启用引脚为 0 时）输出的每一位应该是什么。 选项包括零和浮动； 在后一种情况下，输出实际上与任何其他端口断开。
包括启用吗？
当该属性为 yes 时，该组件具有启用输入。该属性主要用于支持使用不提供启用输入的旧版本 Logisim 构建的电路。
