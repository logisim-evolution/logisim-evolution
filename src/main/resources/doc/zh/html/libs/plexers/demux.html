解复用器

库： 多路解复用器
引入版本： 2.0 Beta 11
外观：

行为

将西侧输入复制到东侧众多输出中的某一路；南侧选择输入的取值决定了输出到哪一路。

引脚（假设组件朝东，选择端在下方偏左）

西侧（输入，位宽与“数据位宽”一致）
要被送往东侧某一路输出的数值。
东侧（输出，数量可变，位宽与“数据位宽”一致）
输出端从北侧开始编号为 0。若某一路编号与南侧选择输入的值相同，则该路输出等于西侧输入； 否则其输出为全 0 或全高阻，取决于 三态 属性。 若选择输入中包含任意未指定位，则所有输出均为高阻。
南侧左端（输入，位宽 1）
使能：为 0 时，无论数据与选择输入为何值，所有输出均为高阻。
南侧右端（灰色圆点标记处，输入，位宽与“选择位数”一致）
选择输入：该输入的值决定把西侧输入送到东侧的哪一路输出。

属性

当组件被选中或正在放置时，数字键 1–4 可修改 选择位数， Alt-0–Alt-9 可修改 数据位宽， 方向键可修改 朝向。

朝向
组件方向（决定哪一侧为输出端）。
选择端位置
选择线与使能线相对于组件的位置。
选择位数
南侧选择输入的位宽。解复用器输出数量为 2选择位数。
数据位宽
通过解复用器被路由的数据位宽。
三态
指定未选中的输出应为高阻（对应：是）还是 0（对应：否）。
禁用时输出
使能端为 0 时，各输出位应呈现的值。可选项包括 0 和高阻；若选择高阻，则输出等同于与其它端口断开连接。
启用使能
当该属性为 是 时，组件带有使能输入端。该属性主要用于兼容旧版 Logisim（当时没有提供使能输入）的电路。
