多路复用器

库： 复用器
引入版本： 2.0 Beta 11
外观：

行为

将西侧众多输入中的一路复制到东侧输出；南侧选择输入的取值决定了选择哪一路输入。

引脚（假设组件朝东，选择端在下方偏左）

西侧（输入，数量可变，位宽与“数据位宽”一致）
数据输入端，其中一路会被送往输出。各输入从北侧开始编号为 0。
东侧（输出，位宽与“数据位宽”一致）
输出值等于西侧输入中编号与南侧选择输入值相同的那一路。 若选择输入中包含任意未指定位，则输出高阻。
南侧左端（灰色圆点标记处，输入，位宽与“选择位数”一致）
选择输入：该输入的值决定将哪一路西侧输入送到东侧输出。
南侧右端（输入，位宽 1）
使能：为 0 时，无论数据与选择输入为何值，输出均为高阻。

属性

当组件被选中或正在放置时，数字键 1–4 可修改 选择位数， Alt-0–Alt-9 可修改 数据位宽， 方向键可修改 朝向。

朝向
组件方向（其输出相对于输入的位置）。
选择端位置
选择线与使能线相对于组件的位置。
选择位数
南侧选择输入的位宽。多路复用器的输入数量为 2选择位数。
数据位宽
通过多路复用器传递的数据位宽。
禁用时输出
使能端为 0 时，各输出位应呈现的值。可选项包括 0 和高阻；若选择高阻，则输出等同于与其它端口断开连接。
启用使能
当该属性为 是 时，组件带有使能输入端。 该属性主要用于兼容旧版 Logisim-evolution 中未提供使能输入的电路。
