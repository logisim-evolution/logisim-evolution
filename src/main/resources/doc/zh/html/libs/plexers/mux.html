复用器

库: 复用器
介绍: 2.0 Beta 11
外观符号:

行为

将西边的输入复制到东边的输出； 要复制哪个输入是通过南边缘输入接收到的当前值指定的。 我发现将多路复用器视为类似于铁路开关，由选择输入控制是有用的。

（顺便说一句，一些权威机构将其拼写为 Multiplex o r ，但 Multiplex e r 是主要拼写。）

引脚（假设元件朝东，选择底部/左侧）

西边，可变数量（输入，位宽与数据位属性匹配）
数据值，其中之一将被路由到输出。 每个输入数据值均已编号，从北面的 0 开始。
东边（输出，位宽与数据位属性匹配）
输出值将与西边的输入值匹配，其编号与当前通过南边的选择输入接收到的值相同。 如果选择输入包含任何未指定（即浮动）位，则输出完全浮动。
南边缘，灰色圆圈表示的左侧（输入，位宽与 Select Bits 属性匹配）
选择输入：此输入的值确定将西边缘的哪个输入路由到东边缘的输出。
南边，右侧（输入，位宽1）
启用：为 0 时，多路复用器的输出由所有浮动位组成，无论数据和选择输入如何。

属性

当选择或添加元件时，数字“1”到“4”会更改其 选择位 属性，Alt-0到Alt-9会更改其 数据位 属性 ，方向键更改其 Facing 属性。

方向
元件的方向（其输出相对于其输入）。
选择位置
选择线和启用线相对于组件的位置。
选择位
元件南边缘的选择输入的位宽度。 多路复用器的输入数量将为 2 selectBits 。
数据位
通过多路复用器路由的数据的位宽度。
禁用输出
指定当元件被禁用时（即，当启用引脚为 0 时）输出的每一位应该是什么。 选项包括零和浮动； 在后一种情况下，输出实际上与任何其他端口断开。
包括启用吗？
当该属性为 yes 时，该元件具有启用输入。该属性主要用于支持使用不提供启用输入的旧版本 Logisim 构建的电路。
