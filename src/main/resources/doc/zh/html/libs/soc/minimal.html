成功仿真所需的最低要求

要执行片上系统（SOC）仿真，您的电路图中至少需要包含：

  1. 一个带有标记的处理器。
  2. 一条带有
  3. 一个标记有

此外，存储器仿真器和处理器都需要连接到总线上。这可以通过定义相应组件的Connected bus属性来实现。

限制

SOC 系统只能位于一张图纸上，这意味着给定的总线组件仅在其所在的图纸上可见。构建层次化系统时，只能将基础系统放在一张图纸上，而扩展部分（例如 VHDL 扩展或电路）则通过输入引脚和输出引脚连接到基础系统。

仿真速度

必须认识到，该库旨在可视化 SOC 中发生的情况。因此，它绝不提供对 SOC 进行（接近）实时仿真的手段。为此，更合适的系统例如QEMU。

在测试系统上，当系统的所有部分都可见时，测得的最大节拍频率为 3 Hz。禁用一些细节以加快仿真速度后，速度提高到 8 Hz。最佳的仿真速度可以通过使用带有动态元件的超级电路来实现。此时仿真速度提高到约 3 kHz。
