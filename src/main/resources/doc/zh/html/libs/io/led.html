LED

库： 输入/输出
引入版本： 2.1.3
外观：

行为

根据输入值显示 LED 的状态：当输入为 1 时，LED 以其 点亮颜色 属性指定的颜色点亮；当输入为 0 时，LED 熄灭。

（LED 组件与 1 位输出引脚在功能上基本重复，只是外观不同。）

引脚

LED 只有一个引脚，为 1 位输入端口，用于决定 LED 是否以点亮颜色显示（输入为 1 时）或保持熄灭颜色（输入为其它值时）。

属性

当组件被选中或正在放置时，可使用方向键修改其 朝向 属性。

朝向
输入引脚相对于组件的位置。
颜色
当输入值为 1 时用于显示的颜色。
高电平点亮
若为 是，则输入为 1 时 LED 点亮；若为 否，则输入为 0 时 LED 点亮。
标签
与该组件关联的标签文本。
标签位置
标签相对于组件的位置。
标签字体
用于绘制标签文本的字体。
标签颜色
绘制标签文本所使用的颜色。

手形工具行为

无。

文本工具行为

可用于编辑与该组件关联的标签文本。
