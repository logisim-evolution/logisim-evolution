操纵杆

库: 输入/输出
介绍: 2.2.0
外观符号:

行为

用户可以在圆角方形区域内拖动红色旋钮，输出会更新以指示旋钮的当前 x 和 y 坐标。 这是为了模仿经典街机游戏时代已知的操纵杆。

引脚

西边，北针（输出，位宽与位宽属性匹配）
指示旋钮的 x 坐标，被解释为无符号整数，其值永远不会为 0。因此，值 1 表示最左侧，位宽的最大值表示最右侧 。 当旋钮处于静止状态（位于中心）时，该值的位模式为 10...00。
西边，南引脚（输出，位宽与位宽属性匹配）
指示旋钮的 y 坐标，其值范围与 x 坐标引脚相同。 当旋钮拉到顶部时，该输出值为 1，当旋钮拉到底部时，输出为所选位宽的最大值。

属性

当选择或添加组件时，Alt-2 到 Alt-5 会更改其 位宽度 属性。

位宽
用于指示每个旋钮坐标的位数。
颜色
旋钮在屏幕上绘制时的颜色。

Poke 工具行为

在操纵杆区域内按下鼠标按钮将旋钮移动到该位置并更新输出。 拖动鼠标继续移动旋钮并更新输出，使旋钮保持在操纵杆的区域内。 释放鼠标按钮会将旋钮恢复到其静止位置。
