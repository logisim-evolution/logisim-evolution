加法器

库: 运算器
引入版本： 2.0 Beta 11
外观：

行为

该组件将西侧两个输入的数值相加，并在东侧输出和。 它支持级联以扩展位数：进位输入提供一个可参与相加的 1 位进位值（若已指定），进位输出则给出 1 位溢出进位，可接到下一级加法器。

若任一加数包含悬空位或错误位，则组件会进行部分相加：尽可能计算低位结果；在出现悬空/错误位之上的更高位，结果将相应变为悬空位或错误位。

引脚

西侧上端（输入，位宽与“数据位宽”一致）
两个加数之一。
西侧下端（输入，位宽与“数据位宽”一致）
另一个加数。
北侧（标记为 c in，输入，位宽 1）
加到和中的进位值。若该值未知（即悬空），则按 0 处理。
东侧（输出，位宽与“数据位宽”一致）
西侧两个加数与 cin 相加后的结果的低 数据位宽 位。
南侧（标记为 c out，输出，位宽 1）
该次相加得到的进位。若将两输入按无符号数相加后能在 数据位宽 位内表示，则该位为 0；否则为 1。

属性

当组件被选中或正在放置时，Alt-0 到 Alt-9 可修改其 数据位宽 属性。

数据位宽
加数与结果的位宽。
