应用程序首选项

Logisim 支持两类配置选项：应用程序首选项 和 项目选项。应用程序首选项会影响所有打开的项目，而项目选项只作用于某一个项目。本节讨论应用程序首选项；项目选项 将在另一节中介绍。

通过菜单 | 文件 |→| 首选项... |（在 Mac OS 下为 Logisim 菜单）可以查看和编辑应用程序首选项。Logisim 会打开一个包含多个选项卡的窗口；下面分别说明这些选项卡，并介绍如何通过命令行配置首选项。

国际化选项卡
窗口选项卡
布局选项卡
仿真选项卡
实验性选项卡
软件选项卡
FPGA Commander 设置选项卡
