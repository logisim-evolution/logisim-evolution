应用程序首选项

Logisim-evolution 支持两类配置选项： 应用程序首选项 和 项目选项 。 应用程序首选项涉及跨越所有打开项目的首选项，而项目选项特定于该项目。 本节讨论应用程序首选项； 项目选项 在另一节中描述。

您可以通过菜单 | File | → | Preferences... | 查看和编辑应用程序首选项（或者， 在 Mac OS 下，Logisim 菜单），将出现一个带有多个选项卡的窗口。 我们将分别讨论这些选项卡，然后我们将了解如何从命令行配置首选项。
