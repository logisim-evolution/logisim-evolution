命令行选项

您可以通过命令行选项配置 Logisim-evolution 的许多应用程序首选项。 这在单学生计算机实验室中特别有用，您希望 Logisim 每次都为学生启动相同的计算机，无论以前的学生如何配置程序。

整体命令行语法如下。 

java -jar jarFileName [options] [filenames]

命令行上命名的可选附加文件将在 Logisim-evolution 中作为单独的窗口打开。

以下示例以基本配置启动 Logisim-evolution。 

java -jar jarFileName  -plain -gates shaped -locale en

支持的选项包括以下内容。

-plain
-empty
-模板 templateFile

配置 Logisim-evolution 使用的模板。

-gates [ shaped | rectangular ]

配置要使用的门类型。

-locale localeIdentifier

配置要使用的翻译。 截至撰写本文时，支持的区域设置包括：

de German
el Greek
en English
es Spanish
fr French
pt Portugais
ru Russian
it Italian
nl Dutch
jp Japanese
-accents [ yes | no ]

这仅与使用 7 位 ASCII 字符集之外的字符的语言相关； 这将包括使用重音字符的语言，但不包括英语。 如果 否 ，则 7 位 ASCII 字符集之外的字符将替换为适合该语言的等效字符； 这对于不能很好支持此类字符的 Java/OS 组合非常有用。

-clearprops

在启动时清除所有应用程序首选项，因此 Logisim-evolution 的行为就像第一次在主机系统上执行一样。

-nosplash

隐藏初始 Logisim-evolution 启动屏幕。

-help

显示命令行选项的摘要。

-version

显示 Logisim-evolution 版本号。

-analyze

显示组合分析菜单。

-load filename

将图像加载到 RAM 中（仅适用于 -tty）

-sub file1 file2

通过将库文件 (file1) 替换为库文件 (file2) 来加载文件。 更多信息请参见 替换库

-tty [ table | speed | tty | halt | stats ]

无需 GUI 即可运行。 更多信息请参阅 其他验证

-testvector circuitname vectorfile projectfile

根据 向量文件 中的向量运行 项目文件 中的 电路名称 测试。 更多信息请参见 窗口文本向量

-geom WxH | WxH+X+Y

打开 Logisim-evolution，其中窗口尺寸根据参数 W（宽度）和 H（高度）确定，或者左上角根据参数 X、Y 定位。
Example: -geom 600x400+100+100
