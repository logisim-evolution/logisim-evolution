命令行选项

可以通过命令行选项配置 Logisim 的许多应用程序首选项。这在学生轮流使用的实验室计算机上特别有用：每次学生启动 Logisim 时，都可以获得相同配置，不受上一位使用者修改过的设置影响。

命令行的一般语法如下。 

java -jar jarFileName [options] [filenames]

命令行中指定的可选附加文件将作为单独的窗口在 Logisim 中打开。

下面的示例会以基础配置启动 Logisim。 

java -jar jarFileName  -plain -gates shaped -locale en

支持的选项包括以下内容。

-plain
-empty
-template templateFile

配置 Logisim 使用的模板。

-gates [shaped|rectangular]

配置使用哪种类型的门。

-locale localeIdentifier

配置界面语言。目前支持的语言环境包括：

de 德语
el 希腊语
en 英语
es 西班牙语
fr 法语
pt 葡萄牙语
ru 俄语
it 意大利语
nl 荷兰语
ja 日语
pl 波兰语
zh 中文
-accents [yes|no]

此选项仅与使用 7 位 ASCII 字符集以外字符的语言相关；这包括使用重音字符的语言，但不包括英语。如果为 no，则 7 位 ASCII 字符集以外的字符会被替换为适合该语言的等效字符；这对于不支持此类字符的 Java/OS 组合很有用。

-clearprops

在启动时清除所有应用程序首选项，因此 Logisim 将表现得像首次在主机系统上执行一样。

-nosplash

隐藏 Logisim 初始启动画面。

-help

显示命令行选项摘要。

-version

显示 Logisim 版本号。

-analyze

显示组合分析菜单。

-load filename

将镜像文件加载到 RAM 中（仅与 -tty 一起使用）。

-sub file1 file2

加载项目时，将库文件 file1 替换为库文件 file2。更多信息见替换库。

-tty [table|speed|tty|halt|stats]

以无 GUI 模式运行。更多信息见其他验证。

-testvector circuitname vectorfile projectfile

基于 vectorfile 中的向量，在 projectfile 中运行 circuitname 测试。更多信息见测试向量窗口。

-geom WxH|WxH+X+Y

打开 Logisim 时，窗口尺寸根据参数 W（宽度）和 H（高度）设置，或根据参数 X、Y 定位左上角。
示例：-geom 600x400+100+100
