模拟菜单

自动传播 Ctrl-E

如果选中，查看的电路将是“实时”的：也就是说，通过电路传播的值将随着电路的每次戳或更改而更新。

如果检测到 电路振荡 ，菜单选项将自动取消选中。

单步传播 Ctrl-I

将模拟向前推进一步 。 例如，信号可能最终在一个步骤中进入门，但门在下一个模拟步骤之前不会显示不同的信号。 为了帮助识别整个电路中的哪些点发生了变化，任何值发生变化的点都用蓝色圆圈表示； 如果子电路包含任何已更改的点（或其子电路，递归地），则它将用蓝色轮廓绘制。

重置模拟 Ctrl-R

清除有关当前电路状态的所有内容，就像您刚刚再次打开文件一样。 如果您正在查看子电路的状态，则整个层次结构都会被清除。

去州

当您通过弹出菜单 深入研究子电路的状态 时， | Go Out To State | 子菜单列出了当前查看的电路状态上方的电路。 选择其中之一会显示相应的电路。

进入状态

如果您已深入研究了子电路的状态，然后又退出，则此子菜单 | Go In To State | 会列出当前电路下方的子电路。 选择其中一个电路会显示相应的电路。

手动刻度半周期 Ctrl-T

使用逐步模式时，将模拟中的时钟前进一格。 设置高/低持续时间为 1 个刻度的时钟将前进半个周期。
当您想要手动前进时钟时，这可能很有用，特别是当时钟与您的时钟不在同一电路中时。 目前正在查看。

手动刻度完整周期 F2

与之前的功能相同，但前进了两个刻度。 设置高/低持续时间为 1 个刻度的时钟将推进一个完整的周期。
该项目的其他时钟按照其参数的比例同时推进。

自动刻度已启用 Ctrl-K

开始自动计时。 仅当电路包含任何时钟设备（在接线库中）时，这才会产生影响。 默认情况下该选项处于禁用状态。

自动滴答频率

允许您选择刻度发生的频率。 例如，8 Hz 表示每秒会出现八次滴答声。 刻度是时钟速度的基本测量单位。

Example: Tic hight = 2, Tic low = 2, Clock = tic frequency / ( 2 + 2)

注意： 时钟周期速度将慢于滴答速度：最快的时钟将具有一个向上滴答周期和一个向下滴答周期； 如果滴答频率为 8 Hz，则此类时钟的向上/向下循环率为 4 Hz。

时序图

进入 记录模块 ，该模块有助于在仿真过程中自动记录和保存电路中的值。

测试向量...

测试向量窗口 借助显示电路输入和输出的测试向量文件，您可以检查电路。

程序集查看器

汇编查看器 窗口显示存储在寄存器中的地址值以及该地址处的汇编语言指令。
