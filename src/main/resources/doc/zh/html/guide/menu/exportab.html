导出图像

选择 | 导出图像... | 后，Logisim 会显示一个包含四个选项的对话框。

配置导出

  • 电路： 在此列表中选择一个或多个要导出为图像文件的电路。（空电路不显示为选项。）
  • 图片格式： 可以创建 PNG、GIF、JPEG、TikZ 和 SVG 文件。通常建议使用 PNG：GIF 格式较旧，JPEG 又主要面向照片，会在电路图像中引入伪影。
  • 缩放系数：使用此滑块调整导出图像的缩放比例。
  • 打印视图：图像样式可以选择包含统计信息，或仅包含原理图。此选项也可在“打印视图”中配置，会改变工作区中的显示。

选择确定后，Logisim 会显示文件选择对话框。如果只选择了一个电路，需要指定图像保存到哪个文件。如果选择了多个电路，需要指定保存目录；Logisim 会根据电路名称命名图像文件（例如 main.png）。
