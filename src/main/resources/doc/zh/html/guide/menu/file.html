文件菜单

New Ctrl-N

在新窗口中打开新项目。 该项目最初将是 当前所选模板 的副本。

Merge Ctrl-M

将项目的组件与当前项目合并，两个项目的组件不得具有相似的名称。

Open... Ctrl-O

在新窗口中将现有文件作为项目打开。

Open Recent

在新窗口中打开最近打开的项目，而不提示用户浏览文件选择对话框。

Close Ctrl-Shift-W

关闭与当前查看的项目关联的所有窗口。

Save Ctrl-S

保存当前查看的项目，覆盖文件中先前的内容。

Save As... Ctrl-Shift-S

保存当前查看的项目，提示用户保存到与以前不同的文件中。

Export Image...

创建与电路对应的图像文件。 配置对话框的描述见 导出图像..

Print... Ctrl-P

将电路发送到打印机。 配置对话框在 打印 中进行了描述。

Preferences...

显示 应用程序首选项 窗口。 （在 Mac OS 系统上，这将出现在 Logisim-evolution 菜单中。）

Exit Ctrl-Q

关闭所有当前打开的项目并终止 Logisim-evolution。 （在 Mac OS 系统上，这将在 Logisim 菜单中显示为“退出”。）
