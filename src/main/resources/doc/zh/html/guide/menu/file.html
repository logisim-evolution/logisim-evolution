文件菜单

新建 Ctrl-N

在新窗口中打开一个新项目。该项目最初是当前选定模板的副本。

合并... Ctrl-M

将另一个项目的组件合并到当前项目；两个项目中不能存在同名组件。

打开... Ctrl-O

在新窗口中将现有文件作为项目打开。

打开最近使用的文件

在新窗口中打开最近打开过的项目，无需再次打开文件选择对话框。

关闭 Ctrl-Shift-W

关闭当前项目的所有关联窗口。

保存 Ctrl-S

保存当前正在查看的项目，覆盖文件中原有内容。

另存为... Ctrl-Shift-S

保存当前正在查看的项目，并要求选择一个不同于原文件的保存位置。

导出图像...

创建与电路对应的图像文件。配置对话框在图像导出中说明。

打印... Ctrl-P

将电路发送到打印机。配置对话框在打印中说明。

首选项...

显示应用程序首选项窗口。（在 Mac OS 系统上，此选项会出现在 Logisim 菜单中。）

退出 Ctrl-Q

关闭所有当前打开的项目并退出 Logisim。（在 Mac OS 系统上，此选项将显示为 Logisim 菜单中的“退出”。）
