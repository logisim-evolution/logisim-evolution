JAR 库

使用 JAR 库

Logisim 中的电路组件大致分为两类：一类是在 Logisim 内部由其他组件组合而成的电路，另一类是用 Java 编写的基本组件。前者更容易设计，但不适合复杂的用户交互，运行效率也相对较低。

Logisim 自带较完整的 Java 组件库，也可以加载由用户或第三方编写的额外库。下载库之后，可以在资源管理器窗格中右击项目（最顶端一行），选择“加载库 > JAR 库...”将其导入项目。随后 Logisim 会要求选择 JAR 文件。在某些情况下，还需要输入库的起始类名；这个类名应由库开发者提供。不过，开发者通常会在 JAR 中加入 manifest 清单文件，并用 Library-Class 属性指定主类名，从而免去这一步。
