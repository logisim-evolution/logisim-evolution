学习更多

除了此处提供的一系列示例之外，Logisim-evolution 源代码还提供了大量其他示例，尽管它们并不总是说明对可读性和良好设计的同样关注。

为了最大程度地移植到未来版本，您应该尽可能坚持使用 ... instance 、... data 和 ... 中的类 工具包。 当然，您可以使用其他软件包的 API，但它们更容易受到 Logisim 未来版本中的更改的影响。

我通常愿意回答偶尔的帮助请求。 当然，错误报告和改进建议总是受欢迎的。

分配

您可以不受限制地自由分发您开发的任何 JAR。 但是，如果您的部分作品源自 Logisim 源代码（根据 GPL 发布）的部分内容，则 GPL 限制确实适用。 从 用户指南 本节中的示例代码派生的代码不会产生此类限制； 这些示例是根据 MIT 许可证发布的。

如果您想与其他 Logisim 用户共享您的库，我将很乐意通过 Logisim 网站提供指向托管网页或 JAR 文件本身的链接。 如果您认为您的库应该内置到基本 Logisim 版本中，那么我欢迎您的建议，并且我将很高兴感谢您在 Logisim 版本（包括工作）中的贡献。
