了解更多

除了这里提供的示例序列之外，Logisim 源代码还提供了大量额外的示例，尽管它们并不总是体现出对可读性和良好设计的同等关注。

为了尽量兼容未来版本，应优先使用 ...instance、...data 和 ...tools 包中的类。其他包的 API 也可以使用，但它们在 Logisim 后续版本中更可能发生变化。

偶尔的求助请求通常可以得到回应；错误报告和改进建议也始终欢迎。

分发

自行开发的 JAR 文件可以不受限制地分发。但是，如果作品中的部分内容源自 Logisim 源代码（按 GPL 发布），则仍需遵守 GPL 的限制。从本用户指南这一部分的示例代码派生作品不会触发这些限制；这些示例按 MIT 许可证发布。

如果希望与其他 Logisim 用户共享库，可以通过 Logisim 网站链接到托管网页或 JAR 文件本身。如果认为某个库应纳入 Logisim 基础发行版，也可以提出建议；若后续版本包含该成果，发布说明中会注明相应贡献。
