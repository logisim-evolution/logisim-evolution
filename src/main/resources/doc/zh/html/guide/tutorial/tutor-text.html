第3步：添加文本

无需向电路添加文本即可使其工作。 但如果您想向某人（例如老师）展示您的电路，那么一些标签有助于传达电路不同部分的用途。

选择 "Text" 工具 ( )。 您也可以通过单击输入引脚并开始输入文本，快捷的为其添加标签。 （直接单击输入引脚比单击文本所在的位置更好，因为这样标签将随引脚移动。）您可以对输出引脚执行相同的操作。 或者您可以单击任何也有文本的位置并开始键入新的内容对其进行修改，同样也可以拖动改变其位置。

许多组件接受标签，例如，如果您单击其中一扇门，您还可以分配标签

可以通过其他几种方式修改标签。

    通过使用 "Edit" 工具 双击组件( ).
    通过点击 "Text" 工具 ( ).
    通过编辑属性 "label" 在 属性表窗口 .
