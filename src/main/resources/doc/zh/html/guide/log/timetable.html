时序图窗口

此窗口用于显示先前所选信号的时序图。窗口包含菜单、工具栏、两个显示区域以及按钮。

| 窗口 | 菜单仅包含一个选项，用于指定窗口是保持在其他窗口之上显示，还是可以进入后台。默认情况下，它保持在最上层。

工具栏有五个按钮。

  启动或停止仿真。等效于 | 仿真 | | 自动传播 | 菜单。
  将仿真向前推进一步。等效于 | 仿真 | | 单步传播 | 菜单。
  启用或禁用 sysclk 时钟。等效于 | 仿真 | | 启用自动节拍 | 菜单。
  将时钟 sysclk 前进半个周期。等效于 | 仿真 | | 步进 1 个节拍（Tick） | 菜单。
将时钟 clk 前进一个周期或半个周期。等效于 | 仿真 | | 'Clk' 单步节拍 | 菜单。

显示区域右侧有三列：第一列显示信号类型的图标（总线或单线），第二列是名称，第三列是光标（图中的红线）处的信号值。可以用鼠标拖动光标。右键单击总线类型信号所在行会弹出一个包含两个选项的菜单：

  • | 格式 |用于选择信号显示进制（二进制、十六进制、八进制、有符号、无符号）。
  • | 展开 |在下方展开显示总线中各个信号的时序图。

按钮： 窗口底部有三个按钮。

  • 加载文件 加载并显示先前保存文件中的结果。此操作不会影响当前仿真设置。
  • 导出... 将数据导出到文本文件。需要在文件选择窗口中指定文件名。文件格式很简单：第一行包含由制表符分隔的列名；第二行显示信号和时间刻度（如果已选择）；后续行显示由制表符分隔的信号值。第三行是初始化，之后每一行代表时钟 sysclk 的半个周期。 
    sysclk     X       a       b       c
clk:X:1:Hz
0       11      0       0       0       
0       11      0       0       0       
1       01      1       1       0       
0       01      0       1       0       
1       10      1       0       1       
0       10      0       0       1       
1       00      1       1       1       
0       00      0       1       1
  • 导出为图片 将时序图导出为 PNG 图像。需要在文件选择窗口中指定文件名。
