文件选项卡

“文件”选项卡允许您指定应将日志放置到的文件。

顶部是文件日志记录是否正在进行的指示器以及用于启用或禁用它的按钮。 （请注意，只有在下面选择文件后才能启用它。）该按钮允许您暂停和重新启动文件输入。 当您在项目窗口中切换以查看另一个模拟时，文件记录会自动停止； 如果您返回到原始记录并希望继续记录，则需要使用顶部的按钮手动重新启用文件记录。

中间是一个指示符，指示正在记录到哪个文件。 要更改它，请使用选择...按钮。 选择文件后，文件记录将自动开始。 如果您选择一个预先存在的文件，Logisim-evolution 将询问您是否要覆盖该文件或将新条目附加到末尾。

在底部，您可以控制是否应将标题行放入文件中，以指示所选内容中的项目。 如果添加标题行，则每当选择更改时，都会将新的标题行放入文件中。

文件格式

条目以制表符分隔格式放入文件中，与“表”选项卡下显示的内容密切对应。 （一个区别是任何标题行都会给出位于子电路中的组件的完整路径。）格式有意简单，以便您可以将其输入另一个程序进行处理，例如 Python/Perl 脚本或电子表格程序。< /p>

为了让脚本可以在 Logisim 运行的同时处理文件，Logisim 将每 500 毫秒将新记录刷新到磁盘上。 请注意，Logisim 也可能在模拟过程中间歇性地关闭并稍后重新打开文件，特别是在几秒钟后没有添加任何新记录的情况下。
