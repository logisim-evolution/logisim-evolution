文件选项卡

文件选项卡用于指定存放时序记录的文件。

顶部有一个指示器，显示文件记录是否正在进行，旁边还有一个用于启用或禁用文件记录的按钮。（请注意，在下方选择文件之前，无法启用文件记录。）可以用该按钮暂停和重新开始文件记录。在项目窗口中切换到另一个仿真时，文件记录会自动停止；如果返回到原来的仿真并希望继续记录，则需要使用顶部的按钮手动重新启用文件记录。

中间有一个指示器，显示正在写入的文件。要更改它，请使用“选择...”按钮。选择文件后，文件记录会自动开始。如果选择了已存在的文件，Logisim 会询问是覆盖该文件，还是将新条目追加到文件末尾。

在底部，可以控制是否在文件中写入标题行，以说明所选内容中包含哪些项目。如果启用标题行，那么每当所选内容发生变化时，都会在文件中写入一个新的标题行。

文件格式

条目以制表符分隔的格式写入文件，与“表格”选项卡下显示的内容非常接近。（一个区别是，标题行会给出位于子电路中的组件的完整路径。）该格式有意保持简单，便于交给其他程序处理，例如 Python/Perl 脚本或电子表格程序。

为了让脚本能够在 Logisim 运行的同时处理文件，Logisim 会每 500 毫秒将新记录刷新到磁盘上。请注意，在仿真过程中，Logisim 也可能会间歇性地关闭并随后重新打开文件，尤其是在几秒钟内没有添加任何新记录的情况下。
