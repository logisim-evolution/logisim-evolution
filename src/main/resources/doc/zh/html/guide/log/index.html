信号选择
时序图窗口

为了理解或调试电路，通常需要以可视方式观察不同信号的变化。时序图模块正是为此而设计的；它可以把信号记录为图形，也可以以文本文件中的数值表形式保存。

您可以通过菜单| 仿真 |→| 时序图 |进入记录模块。这将弹出信号选择窗口。

下面这些时钟用于驱动信号显示。仿真中有两种特定的时钟：一个是必需的，命名为sysclk；另一个是可选的辅助时钟，命名为clk。

注意：电路中必须有一个名为sysclk的时钟。它将被时序图模块用作时基。它不必连接到您的电路。原则上它是最快的，并且高、低电平持续时间均为 1 个节拍。
