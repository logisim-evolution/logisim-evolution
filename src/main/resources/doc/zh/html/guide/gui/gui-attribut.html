属性表

许多组件都有属性，这些属性用于配置组件的行为或外观。属性表用于查看和显示组件的属性值。

要选择查看哪个组件的属性，请使用编辑工具（）单击该组件。您也可以右键单击或按住 Control 键单击该组件，然后从弹出菜单中选择“显示属性”。此外，通过手形工具（）或文本工具（）操作组件，也会显示该组件的属性。

下面的屏幕截图展示了在选择异或电路的上方输入引脚并向下滚动查看标签字体属性后的效果。

要修改属性值，请点击该值。修改属性的界面取决于您更改的属性；对于标签字体属性，将出现一个对话框用于选择新字体；但有些属性（如标签）允许您以文本字段形式编辑值，而其他属性（如标签位置）则会显示一个下拉菜单供您选择值。

每种组件类型都有不同的属性集；要了解它们的含义，请参阅库参考中的相关文档。

如果您使用编辑工具（）选择了多个组件，则属性表将显示所有选定组件（不包括任何导线）共享的属性。如果选择的组件对于某个属性并非全部具有相同的值，则显示的值将为空白。您可以使用属性表一次性更改所有选定组件的属性值。
