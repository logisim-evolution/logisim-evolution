菜单

本节快速解释每个主要 logisim-evolution 窗口附带的七个菜单。

文件菜单： 此菜单允许您打开、关闭以保存项目。 它允许您访问首选项并允许您退出该程序。

编辑菜单： 主要用于编辑、复制、删除、复制、移动绘图元素。

项目菜单： 允许您管理库、更改导航窗格中的顺序以及更改此窗格的外观。 它允许访问项目首选项。

模拟菜单： 控制模拟，开始，停止，进入步进模式，启动时钟。

FPGA菜单： 控制与FPGA开发板、FPGA板编辑器、综合和下载的链接

窗口菜单： 最小化、展开、关闭窗口。 转至组合分析窗口（如果启用）和应用程序首选项。

帮助菜单： 转到教程、参考指南或库参考。

您可以在 专门讨论此主题的页面 中找到有关菜单的完整信息。
