菜单

本节快速解释每个主要 Logisim 窗口附带的七个菜单。

文件菜单：用于打开、关闭、保存项目，也提供首选项入口和退出程序的命令。

编辑菜单：主要用于编辑、剪切/复制、删除、创建副本和移动绘图元素。

项目菜单：用于管理库、更改导航窗格中的顺序以及更改此窗格的外观，也提供项目选项入口。

仿真菜单： 控制仿真：启动、停止、进入单步模式以及控制时钟。

FPGA 菜单： 控制与 FPGA 开发板的连接、FPGA 板编辑器、综合与下载。

窗口菜单：最小化、最大化、关闭窗口。转到组合分析窗口（如果启用）和应用程序首选项。

帮助菜单：转到教程、参考指南或库参考。

关于菜单的完整信息，请参阅菜单参考。
