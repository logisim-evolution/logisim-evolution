菜单

本节快速解释每个主要 Logisim 窗口附带的七个菜单。

文件菜单：该菜单允许您打开、关闭、保存项目。它提供对首选项的访问并允许您退出程序。

编辑菜单：主要用于编辑、复制、删除、复制、移动绘图元素。

项目菜单：允许您管理库、更改导航窗格中的顺序以及更改此窗格的外观。它提供对项目首选项的访问。

仿真菜单： 控制仿真：启动、停止、进入单步模式、启动时钟。

FPGA 菜单： 控制与 FPGA 开发板的连接、FPGA 板编辑器、综合与下载。

窗口菜单：最小化、最大化、关闭窗口。转到组合分析窗口（如果启用）和应用程序首选项。

帮助菜单：转到教程、参考指南或库参考。

您可以在专门介绍该主题的页面中找到关于菜单的完整信息。
