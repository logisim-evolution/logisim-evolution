上一节: 图形界面

画布

canvas确实是Liogisim-evolution中最明显的部分，您可以在其中绘制电路及其符号以最终模拟操作。

它仅由几个元素组成：

水平和垂直滚动条：它们的工作方式与大多数程序相同，只需拖动滑块即可将视图移动到绘图的其他部分。 还可以通过键盘或鼠标通过以下组合来操作它们：

  • 鼠标滚轮或按键向上/向下箭头：垂直滚动
  • Shift-鼠标滚轮或按键右/左箭头：水平滚动
  • 按键PgUp：转到页面顶部
  • 键PgDn：转到页面底部

范围指示器当原理图大于指示器的显示区域时显示。 显示在角落和/或侧面的参考线指示绘图延伸的方向。 以下是页面顶部指标的示例。

居中按钮 它将图表的中心放置在工作表面的中心。

缩放：左下角是缩放系数的控件。 您可以通过拖动光标或使用速率左侧和右侧的两个按钮之一或使用工作区域中的 Ctrl - 鼠标滚轮 来更改它。

按钮 100% ：适合缩放以按实际大小显示图表。
按钮自动：适合缩放，以便整个图表可见。
网格按钮：: 在缩放右侧，交替切换模式“网格显示” 模式“隐形网格”。
