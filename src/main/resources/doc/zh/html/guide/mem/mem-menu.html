弹出菜单和文件

弹出菜单

存储器的弹出菜单除了所有组件通用的选项外，还包括四个选项：

编辑内容：
打开十六进制编辑器以编辑存储器的内容。
清除内容：
将存储器中的所有值重置为 0；对于 RAM，则根据项目选项中的参数重置为 0 或随机值。
加载映像……：
将存储器中的所有值重置为文件中的值。文件格式会自动识别；如果无法识别，则会打开存储器文件面板窗口，以便选择合适的格式。
保存映像……：
使用下述格式之一，将存储器中的所有值存入文件。

文件格式

  • v2.0 raw 格式 Logisim 的历史格式，支持“游程长度”编码。
  • v3.0 十六进制字格式（普通） 与 v2.0 的字结构格式接近。
  • v3.0 十六进制字格式（带地址） 与普通格式类似，但可以为文件的每一行定义起始地址。
  • v3.0 十六进制字节格式（普通大端）
    v3.0 十六进制字节格式（普通小端） 按字节流组织。
  • v3.0 十六进制字节格式（带地址大端）
    v3.0 十六进制字节格式（带地址小端） 按字节流组织，并可以定义地址。
  • 二进制数据（大端）
    二进制数据（小端） 存储器映像，不使用十六进制编码。
  • ASCII 字节转义（大端）
    ASCII 字节转义（小端） ASCII 编码文件，对不可打印字符使用类似 C 语言的转义码。
