弹出菜单和文件

弹出菜单

除了所有组件共有的选项之外，内存弹出菜单还包括四个选项：

编辑内容：
打开 十六进制编辑器 来编辑内存内容。
清除内容：
将内存中的所有值重置为 0，将 RAM 重置为 0，或者根据 项目选项 中的参数随机重置。
加载图像...：
将内存中的所有值重置为文件中找到的值。 文件格式会自动识别，如果不是这种情况，则会打开 窗口内存文件面板 以进行适当的选择。
保存图像...：
使用下述格式之一将内存中的所有值存储在文件中。

文件格式

  • v2.0 raw logisim-evolution 的历史版本允许“run.length”编码。
  • v3.0 十六进制单词纯文本 接近 v2.0 单词结构。
  • v3.0 十六进制字地址 与以前一样，可以为文件的每一行定义起始地址。
  • v3.0 十六进制字节纯大端
    v3.0 十六进制字节纯小端 以字节流形式组织。
  • v3.0 十六进制字节寻址大端
    v3.0 十六进制字节寻址小端 像以前一样以字节流组织，可以定义地址。
  • 二进制数据大尾数
    二进制数据小尾数 未编码的十六进制内存图像。
  • Ascii 字节转义大端
    Ascii 字节转义小端 带有转义代码的 ASCII 编码文件，如 C 中的不可打印字符。
