Ascii 字节转义大尾数

带转义码的 ASCII 编码，除不可打印和非标准字符外，数据将逐字节记录。 0x20 - 0x7E 范围之外的字符将被忽略。 对于这种类型的字符，必须使用十六进制转义代码 \x... 或常用代码之一 \0 \a \b \t \n \v \f \r。 

\0\0hello world\r\nGood morning!\r\n
\xCD\xCD\xCD\xCD\xCD\0\0\0\0

在大端模式下，内存被视为字节序列，无论其字的大小如何，从左到右先是高字节，然后是低字节。 您可以在 v3.0 十六进制字节寻址大端 中找到插图

Ascii 字节转义小尾数

此格式与前一种格式相同，具有相同的特征。 在小端模式下，内存被视为字节序列，从右到左依次为最低有效位和最高有效位。 您可以在 v3.0 十六进制字节寻址小端 中找到插图
