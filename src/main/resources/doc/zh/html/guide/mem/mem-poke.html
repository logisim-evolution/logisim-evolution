尽管如此，要查看和编辑电路中的值，Poke Tool 有两种操作模式：您可以编辑显示的地址，也可以编辑单个值。

地址选择

要编辑显示的地址，请在显示矩形外部单击。 Logisim-evolution 将在顶部地址周围绘制一个红色矩形。

  • 键入十六进制数字将相应地更改顶部地址。

  • 键入 Enter 键 将向下滚动一行。

  • 键入 退格键 将向上滚动一行。

  • 键入 空格键 将向下滚动一页（四行）。

数据修改

要编辑特定值，请单击显示矩形内的值。 Logisim-evolution 将在该地址周围绘制一个红色矩形。

  • 键入十六进制数字将更改当前正在编辑的地址处的值。

  • Enter 键 或 Ctrl+向下箭头 会将光标向下滚动一行。

  • 退格键 或 Ctrl+向左箭头 会将光标向后滚动一个单元格。

  • 空格键 或 Ctrl+右箭头 会将光标向前滚动一个单元格。

  • 鼠标滚轮 或 Ctrl+向上箭头 、 Ctrl+向下箭头 移动 光标向上/向下移动一行。
