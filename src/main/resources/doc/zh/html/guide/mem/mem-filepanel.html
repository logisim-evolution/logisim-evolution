存储器文件面板

当文件识别无法确定文件结构类型时，此窗口会打开。它包含一系列复选框，用于指定转换选项。两个十六进制显示选项卡显示文件数据，文本框显示检测到的错误。

单选按钮用于选择上方显示的文件结构类型之一。选择某种类型后，相应的复选框会启用，用于指定具体选项。
这些格式的详细说明见以下页面：

  • v2.0 raw 格式
  • v3.0 十六进制字格式（普通/带地址）
  • v3.0 十六进制字节格式（大端/小端）
  • 二进制数据（大端/小端）
  • ASCII 字节转义（大端/小端）

十六进制显示面板用于查看 Logisim 如何解码文件，并将解码结果与原始文件进行比较。

下方会显示解码报告以及发现的任何错误。
