创建电路

每个 Logisim-evolution 项目实际上都是一个电路库。 在最简单的形式中，每个项目只有一个电路（默认情况下称为 main ），但添加更多电路很容易：单击 小菜单栏 位于导航面板上方，或者 右键单击 导航面板中树的根部，或者选择 | Project | → | Add Circuit... | ，然后为所需的新电路输入您喜欢的任何名称创造。

注意： 电路名称必须以字母（A..Z、a..z）开头，可以包含符号（.、_）和数字。
