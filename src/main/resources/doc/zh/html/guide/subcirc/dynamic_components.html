动态组件

动态组件用于观察或操作隐藏在上层子电路内部的组件。例如：您在名为 MyNiceCircuit 的电路中放置了一个 LED，然后在名为 MyToplevel 的电路中把 MyNiceCircuit 作为子电路使用。在 MyToplevel 中仿真时，这个 LED 会被隐藏，因此无法直接看到 MyNiceCircuit 中的 LED 是否点亮。动态组件可以把 LED 的图像放到 MyNiceCircuit 的子电路符号上，从而解决这个问题。要向 MyNiceCircuit 的子电路符号添加动态组件，先打开 MyNiceCircuit 的外观编辑器，然后点击 工具图标。随后会打开一个窗口，列出所有可用的动态组件，您可以选择要在子电路符号外观中显示的组件。动态组件具有递归特性：如果电路具有层次结构，它会显示当前子电路下方所有可用的动态组件。
