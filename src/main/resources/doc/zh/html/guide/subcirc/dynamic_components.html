动态组件

动态组件提供了观察/操作隐藏在更高级别的子电路中的组件的方法。 示例：您已在名为 MyNiceCircuit 的电路中放置了一个 LED。 现在，您使用 MyNiceCircuit 作为名为 MyToplevel 的电路中的子电路。 在 MyToplevel 中进行模拟会隐藏您的LED，因此您无法查看 MyNiceCircuit 中的LED是否点亮。 动态组件可以在 MyNiceCircuit 的子电路符号上放置 LED 图像，从而提供帮助。 要将动态组件添加到 MyNiceCircuit 的子电路符号中，您必须首先打开 外观编辑器 电路 MyNiceCircuit 。 然后单击 -tool icon。 将打开一个窗口，列出所有可用的动态组件，您可以选择要在子电路符号的外观中显示的组件。 请注意，动态组件功能是递归属性。 这意味着，如果您有分层结构，它将显示当前子电路下所有可用的动态组件。

当前以下工具支持动态组件：


LED-tool
