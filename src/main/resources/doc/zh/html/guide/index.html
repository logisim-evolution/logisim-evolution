Logisim-evolution 用户指南

Logisim-evolution 是一种用于设计和模拟数字逻辑电路的教育工具。 凭借其简单的工具栏界面和构建电路的模拟，它足够简单，可以帮助学习与逻辑电路相关的最基本概念。 Logisim-evolution能够从较小的子电路构建较大的电路，并通过单击鼠标拖动来绘制线束，因此可以使用（并且已经使用）出于教育目的设计和模拟整个 CPU。

世界各地学院和大学的学生将 Logisim 用于多种目的，包括：

  • 普通教育计算机科学调查模块
  • 大二计算机课程的一个单元
  • 整个学期的高年级计算机架构课程
