Logisim-evolution 用户指南

Logisim-evolution 是一款面向教学的数字逻辑电路设计与仿真工具。它提供简单的工具栏界面，并能在搭建电路的同时即时仿真，因此适合学习逻辑电路的基础概念。它也支持用较小的子电路搭建更大的电路，并可一次拖动绘制多位线束；因此在教学场景中，Logisim 也可用于设计和仿真完整 CPU。

世界各地高校的学生在多种课程中使用 Logisim，例如：

  • 通识教育计算机科学概览中的一个模块
  • 大二年级计算机组成原理课程中的一个单元
  • 高年级计算机体系结构课程中的整个学期

《Logisim 用户指南》，也就是当前文档，是 Logisim 功能的官方参考。第一部分由一系列章节组成，介绍 Logisim 的主要功能。这些章节可以按顺序从头读到尾，用来系统了解 Logisim 的重要功能。
