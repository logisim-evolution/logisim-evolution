Logisim 包含对从命令行执行电路的基本支持。这旨在帮助进行电路设计的脚本化验证，并帮助教师对学生解决方案进行自动化测试。

本节首先介绍如何从命令行执行电路。示例假设下面的电路已保存在名为 adder-test.circ 的文件中。它使用一个两位加法器作为子电路，并通过计数器遍历全部 16 种可能输入。

构建此电路后，从命令行执行 Logisim，提供项目文件名，并为 -tty 选项指定 table 参数。

java -jar logisim-filename.jar adder-test.circ -tty table

Logisim 不会显示任何窗口，而是加载电路并开始执行，在完成每个节拍之间的传播的同时，以尽可能快的速度触发时钟。每次传播完成后，Logisim 会加载输出引脚的当前值；如果任何值与上一次传播相比发生了变化，则所有值都会以制表符分隔的格式显示。如果有一个输出引脚标有特殊词 halt（区分大小写），其输出不会显示——但一旦传播完成后该引脚的值达到 1，Logisim 就会结束仿真。

在这个示例中，Logisim 显示下表。前两个输出引脚对应两位加法器的输入 a 和 b，因此它们构成输出的前两列；另一个输出引脚 sum 对应两位加法器的输出，因此位于第三列。列的顺序按照电路中从上到下的顺序自左向右排列。
