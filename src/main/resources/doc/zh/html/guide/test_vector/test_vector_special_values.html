特殊值

测试向量格式支持两种特殊值，以实现更灵活的测试：

  • <DC>（无关项）：当用于输出引脚时，无论实际输出为何，此值始终通过。当您在测试中不关心某些输出时，这很有用。不区分大小写（<dc> 同样有效）。
  • <float>：当用于输入引脚时，驱动一个浮空（高阻）值。当用于输出引脚时，期望输出为浮空（未知值）。不区分大小写（<FLOAT> 同样有效）。

以下是一个使用特殊值的示例：

# Test with don't care and floating values
Input   Enable Output  <set> <seq>
1       1      <DC>    0     0
<float> 0      <float> 0     0

在第一个测试中，输出标记为无关项，因此任何输出值都会通过。在第二个测试中，输入为浮空，输出也预期为浮空。
