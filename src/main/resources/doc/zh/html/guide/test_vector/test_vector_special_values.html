特殊值

测试向量格式支持两个特殊值，以实现更灵活的测试：

  • <DC>（无关）：当用于输出引脚时，无论实际输出如何，此值始终通过。当您在测试期间不关心某些输出时，这很有用。不区分大小写（<dc> 也可以）。
  • <float>：当用于输入引脚时，这会驱动一个浮空（高阻抗）值。当用于输出引脚时，这期望输出为浮空（UNKNOWN 值）。不区分大小写（<FLOAT> 也可以）。

以下是使用特殊值的示例：

# 使用无关和浮空值的测试
Input   Enable Output  <set> <seq>
1       1      <DC>    0     0
<float> 0      <float> 0     0

在第一个测试中，输出被标记为无关，因此任何输出值都会通过。在第二个测试中，输入是浮空的，输出也期望是浮空的。
