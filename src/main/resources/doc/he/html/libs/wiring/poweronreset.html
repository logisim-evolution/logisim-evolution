Power on Reset

Library: Wiring
Introduced: ...
Appearance:

Behavior

The POR component emits a high signal during a time defined by the property POR-Time at the start of the simulation. This happens either when the project is opened by logisim or when you use the Restart simulation menu. In step-by-step mode Simulation menu | Step-by-step simulation the transition will take place after the delay but in synchronization with the pressing of the key (Ctrl-I ).

Pins

This component has only one one-bit pin whose value will represent the current state of the reset. The location of this pin is specified in the attribute Facing

Attributes

When the component is selected or added, the arrow keys change its Facing attribute. The side of the drawn component where its output pin should be located. The time in seconds before the signal goes down to the low level (0)

Facing
The side of the drawn component where its output pin should be located.
POR-Time
The time in seconds before the signal goes down to the low level (0)
