Do not connect

Library: Wiring
Introduced:  3.2 1 in wiring library
Appearance:

Behavior

Constantly generates a floating or high impedance state.

Pins

A Do Not Connect component has only one output pin whose bit width corresponds to the "Data Bits" attribute

Attributes

When the component is selected or being added, Alt-0 through Alt-9 alter its Data Bits attribute. Quick use of the keys allows the entry of larger values for example Alt-0ALT-1 ALT-2 gives 12.

Data Bits
The bit width of the value placed onto the wire.

Poke Tool Behavior

None.

Text Tool Behavior

None.
