חלון Test Vector

חלון Test Vector דומה ללשונית טבלה של רישום חַלוֹן. אתה יכול לטעון וקטור בדיקה מקובץ, ו-Logisim יתחיל הפעלת בדיקות במעגל הנוכחי. כמו חלון הרישום, שם הוא רק חלון וקטור בדיקה אחד עבור הפרויקט, והטבלה תעשה זאת שנה כדי לשקף את המעגל שמדמה בפרויקט חַלוֹן. שים לב, עם זאת, מודול Test Vector פועל בנפרד עותק של סימולטור המעגל, ולכן אינו מפריע, והוא לא מושפע מהסימולציה בחלון הפרויקט.

כל פלט שגוי יסומן באדום. העבר את העכבר מעל התיבה האדומה כדי לראות מה הפלט היה צריך להיות, על פי וקטור בדיקה. שורות עם פלטים שגויים ממוינים לראש ה- חלון.

פורמט הקובץ פשוט. אתה יכול להשתמש במודול רישום (עם "כלול שורת כותרת" נבחר בכרטיסיית פלט הקובץ) כדי להתחיל, מכיוון שברוב המקרים מודול Logging מוציא את אותו הפורמט שבו נעשה שימוש על ידי מודול Test Vector.

הנה קובץ וקטור לבדיקה לדוגמה:. 

# my test vector for add32
A[32] B[32] C[32] Cin Cout
00000000000000000000000000000000        00000000000000000000000000000000        00000000000000000000000000000000        0       0
-2       0x00000005    3       0       0
0        0o0003        3       0       0

מתעלמים משורות ריקות. כל דבר העוקב אחר תו '#' הוא a תגובה. השורה הראשונה שאינה ריקה, ללא הערות, מפרטת את השם של כל אחד פין קלט מעגל והרוחב שלו (אם > 1), מופרדים על ידי רווח לבן.

שאר השורות מפרטות כל ערך מופרד על ידי רווח לבן. ה הערכים יכולים להיות ב-hex, אוקטלי, בינארי או סימן עשרוני. ערכי hex חייבת להיות קידומת '0x'. לערכים אוקטליים יש קידומת '0o'. בינארי ועשרוני הם be מובחן על ידי מספר הספרות: ערכים בינאריים חייבים תמיד להיות מספרים בדיוק כמו רוחב העמודה; ערכים עשרוניים חייבים תמיד יש פחות, לא צריך להיות אפסים מובילים, וייתכן שיהיה א סימן שלילי.

עבור ערכים הקסדיים, אוקטליים ובינאריים, ספרה של 'x' מציינת ארבע, שלושה, או אחד "לא אכפת" ביטים. אז הערך 101xx הוא חמש סיביות ערך בינארי, כאשר שני הביטים האחרונים לא צוינו, ו-0x1ax5 הוא hex ערך עם ארבע ביטים לא מוגדרים. אי אפשר להשתמש ב"לא אכפת" שכזה בסימון עשרוני.

שורת פקודה: כדי להקל על בדיקות אוטומטיות, ה ניתן להפעיל תכונה וקטור בדיקה משורת הפקודה, באופן הבא:
logisim -test <circuitname> <vector.txt> <project.circ>

הבא: מדריך למשתמש.
