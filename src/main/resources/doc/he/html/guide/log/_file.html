הכרטיסייה קובץ

הכרטיסייה קובץ מאפשרת לך לציין קובץ שלתוכו יומן הרישום להיות ממוקם.

למעלה מופיע אינדיקטור של האם רישום קבצים מתבצע כפתור להפעלה או השבתה. (שים לב שאינך יכול להפעיל אותו עד שנבחר קובץ למטה.) הכפתור מאפשר לך להשהות ו הפעל מחדש את הזנת הקובץ. כאשר אתה עובר בחלון הפרויקט לצפייה סימולציה נוספת, רישום הקבצים נעצר אוטומטית; אם אתה לחזור למקור ולרצות שהרישום ימשיך, תצטרך כדי להפעיל מחדש את רישום הקבצים באופן ידני באמצעות הכפתור בחלק העליון.

באמצע יש אינדיקטור לאיזה קובץ נכנס. ל שנה אותו, השתמש בלחצן בחר... בבחירת קובץ, רישום קבצים יתחיל אוטומטית. אם תבחר קובץ קיים מראש, Logisim ישאל אם אתה רוצה להחליף את הקובץ או להוסיף את החדש ערכים לסוף.

למטה תוכל לקבוע אם יש להציב שורת כותרת ב הקובץ המציין אילו פריטים נמצאים בבחירה. אם שורות כותרת יתווספו, אז תוצב שורת כותרת חדשה בקובץ בכל פעם הבחירה משתנה.

פורמט קובץ

ערכים ממוקמים בקובץ בפורמט מופרד בטאבים תואם באופן הדוק למה שמופיע תחת הכרטיסייה טבלה. (אחד ההבדל הוא שכל שורות כותרת יתנו את הנתיב המלא רכיבים השוכבים במעגלי משנה.) הפורמט פשוט בכוונה ולכן שאתה יכול להזין אותו לתוכנית אחרת לעיבוד, כגון א סקריפט Python/Perl או תוכנית גיליון אלקטרוני.

כדי שסקריפט יוכל לעבד את הקובץ במקביל ללוגיסים פועל, Logisim תשטוף את הרשומות החדשות לדיסק כל פעם 500 אלפיות השנייה שימו לב שגם לוגיסים עשויה להיסגר לסירוגין ומאוחר יותר פתח מחדש את הקובץ במהלך הסימולציה, במיוחד אם מספר שניות חלפו מבלי שנוספו רשומות חדשות.

הבא: מדריך למשתמש.
