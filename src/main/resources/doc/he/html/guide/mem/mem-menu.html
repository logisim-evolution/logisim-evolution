תפריטים וקבצים קופצים

תפריט קופץ

התפריט המוקפץ לזיכרון כולל ארבע אפשרויות בנוסף לאפשרויות המשותפות לכל הרכיבים:

ערוך תוכן:
העלה עורך hex לעריכת תוכן הזיכרון.
נקה תוכן:
מאפס את כל הערכים בזיכרון ל-0 ול-RAM ל-0 או באופן אקראי בהתאם לפרמטרים ב-אפשרויות פרויקט .
טען תמונה...:
מאפס את כל הערכים בזיכרון לערכים שנמצאו בקובץ. פורמט הקובץ מזוהה אוטומטית ו אם זה לא המקרה, חלונית קבצי זיכרון נפתחת עבור חלון מתאים בְּחִירָה.
שמור תמונה...:
מאחסן את כל הערכים בזיכרון בקובץ באמצעות אחד מהפורמטים המתוארים להלן.

פורמט קובץ

  • v2.0 raw גרסה היסטורית של logisim מאפשרת קידוד "run.length".
  • v3.0 מילים hex plain קרוב ל-v2.0 מילה מובנית.
  • v3.0 מילות hex מטופלות כמו קודם עם אפשרות להגדיר כתובת התחלה עבור כל שורה בקובץ.
  • v3.0 hex bytes רגיל big-endian
    v3.0 hex bytes plain little-endian מאורגן בזרמי בתים.
  • v3.0 hex bytes כתובות big-endian
    v3.0 hex bytes כתובות little-endian מאורגן בזרימת בתים כמו לפני עם אפשרות להגדיר כתובת.
  • נתונים בינאריים big-endian
    נתונים בינאריים little-endian תמונה של הזיכרון ללא קידוד בהקסדצימלי.
  • Ascii byte escape big-endian
    Ascii byte escape little-endian קובץ מקודד ASCII עם קוד בריחה, כמו ב-C עבור תווים שאינם ניתנים להדפסה.

הבא: חלונית קבצי זיכרון.
