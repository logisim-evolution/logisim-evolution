Параметры командной строки

Вы можете переопределить многие настройки приложения Logisim-evolution с помощью параметров командной строки. Это особенно удобно для учебных классов или лабораторий, где требуется запускать программу в абсолютно идентичной конфигурации для каждого учащегося, независимо от того, какие настройки оставил предыдущий пользователь.

Общий синтаксис запуска из командной строки выглядит следующим образом: 

java -jar имя_jar_файла [параметры] [файлы_проектов]

Необязательные дополнительные файлы проектов, указанные в конце командной строки, будут открыты в отдельных окнах Logisim-evolution.

Следующий пример запускает программу в базовой конфигурации: 

java -jar имя_jar_файла -plain -gates shaped -locale en

Поддерживаются следующие параметры командной строки:

-plain
-empty
-template файл_шаблона

Задаёт шаблон для использования при создании новых проектов. Вы можете выбрать обычный шаблон по умолчанию (-plain), пустой шаблон (-empty) или указать путь к файлу специального шаблона (-template).

-gates [shaped|rectangular]

Задаёт стиль отрисовки логических элементов: фигурные (shaped, стандарт ANSI) или прямоугольные (rectangular, стандарт IEC/ГОСТ).

-locale идентификатор_языка

Задаёт язык интерфейса программы. На данный момент поддерживаются следующие локали:

de Немецкий
el Греческий
en Английский
es Испанский
fr Французский
it Итальянский
jp Японский
nl Голландский
pl Польский
pt Португальский
ru Русский
-accents [yes|no]

Имеет значение только для языков, использующих символы диакритики, выходящие за рамки стандартного 7-битного ASCII. При значении no такие символы будут заменяться близкими ASCII-эквивалентами, что решает проблемы отображения на некоторых комбинациях Java и операционных систем.

-clearprops

Очищает все сохранённые настройки приложения при старте. Программа запустится так, будто она выполняется на этом компьютере впервые.

-nosplash

Скрывает стартовую заставку (сплэш-экран) программы.

-help

Выводит краткую справку по всем доступным параметрам командной строки.

-version

Отображает текущую версию Logisim-evolution.

-analyze

Открывает инструмент комбинационного анализа без запуска основного графического интерфейса редактора.

-load файл_образа

Загружает образ памяти в ОЗУ/ПЗУ (работает только совместно с флагом -tty). Дополнительную информацию см. в разделе «Другие возможности верификации».

-sub библиотека_1 библиотека_2

Загружает проект с автоматической заменой файлов одной библиотеки (библиотека_1) файлами другой (библиотека_2). Подробнее см. в разделе «Замена библиотек».

-tty [table|speed|tty|halt|stats]

Запускает моделирование без открытия графического интерфейса пользователя. Дополнительную информацию см. в разделе «Другие возможности верификации».

-testvector имя_схемы файл_векторов файл_проекта

Выполняет автоматическое тестирование указанной схемы в файле проекта на основе набора тестовых векторов. Подробнее см. в разделе «Тестовые векторы».

-geom ШиринаxВысота|ШиринаxВысота+X+Y

Задаёт геометрические размеры главного окна программы и координаты его левого верхнего угла на экране при запуске. Пример: -geom 800x600+100+100.

Недокументированные параметры

-test-circuit путь_к_тестовой_схеме
-test-fpga-implementation путь_к_проекту файл_привязки имя_схемы плата
-test-circ-gen входной_файл_схемы выходной_файл_схемы
-questa[yes|no]

Далее: Назад к руководству пользователя.
