Вкладка «Моделирование»

Эта вкладка содержит параметры, определяющие цвета проводников в процессе моделирования логических уровней, а также цвета, используемые для визуализации покрытий на картах Карно.

В верхней части окна настраиваются индивидуальные цвета для каждого из возможных состояний проводников (например, логический 0, логическая 1, неинициализированное значение (U), высокоимпедансное состояние (Z) или ошибка разрядности). Здесь же можно выбрать символы, которые будут отображаться на линиях для индикации их текущего статуса.

В нижней части окна задаются цвета, используемые модулем анализа для выделения групп и покрытий при минимизации логических функций на картах Карно.

Изменение параметров выполняется стандартным образом: при щелчке по кнопке с цветом открывается диалоговое окно системной палитры, где можно выбрать и применить любой желаемый оттенок.

Для быстрого переключения схем предусмотрены две управляющие кнопки:

  • Цвета по умолчанию — сбрасывает все текущие настройки к заводской стандартной цветовой палитре Logisim-evolution.
  • Цвета для дальтоников — автоматически загружает специальную высококонтрастную палитру, оптимизированную для пользователей с нарушениями цветового зрения.
