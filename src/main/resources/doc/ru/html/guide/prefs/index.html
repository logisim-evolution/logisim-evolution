Настройки приложения

В Logisim-evolution параметры конфигурации делятся на две категории: настройки приложения и параметры проекта. Настройки приложения распространяются на все открытые проекты, в то время как параметры проекта уникальны для каждого файла проекта. В этом разделе рассматриваются настройки приложения; параметры проекта описаны в отдельной главе.

Вы можете просматривать и изменять настройки приложения через меню | Файл |→| Настройки... | (или в меню Logisim-evolution на macOS). При этом откроется окно с несколькими вкладками.

  • Вкладка «Шаблон»
  • Вкладка «Язык»
  • Вкладка «Окно»
  • Вкладка «Холст»
  • Вкладка «Моделирование»
  • Вкладка «Экспериментальные»
  • Вкладка «ПЛИС»
  • Параметры командной строки

Далее: Вкладка «Шаблон».
