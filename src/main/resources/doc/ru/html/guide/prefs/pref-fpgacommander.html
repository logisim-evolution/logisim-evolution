Вкладка «ПЛИС»

Эта вкладка содержит настройки уровня приложения, используемые Панелью управления ПЛИС (FPGA Commander).

  • Рабочая папка (Workspace): выбор папки, в которую Панель управления ПЛИС записывает сгенерированные HDL-файлы, скрипты, файлы ограничений (constraints) и результаты работы внешних инструментов.

  • Язык описания аппаратуры: выбор языка описания аппаратуры (HDL), используемого по умолчанию для проектов ПЛИС.

  • Настройки VHDL: управление параметрами генерации VHDL (например, записывать ли ключевые слова языка VHDL заглавными буквами).

  • Цвета редактора плат: настройка цветов, используемых при определении компонентов на изображении платы ПЛИС.

  • Цвета привязки к плате: настройка цветов, используемых при сопоставлении компонентов схемы физическим ресурсам платы ПЛИС.

  • Параметры проверки правил проектирования (DRC): определяет, будут ли отображаться в отчёте Панели управления ПЛИС выбранные предупреждения (например, о тактировании через логические элементы или о неподключённых входах).

Отчёты Панели управления ПЛИС

Область отчётов Панели управления ПЛИС содержит вкладки для информационных сообщений, предупреждений, ошибок и вывода консоли. Поскольку встроенные вкладки отчётов имеют небольшой размер, вы можете дважды кликнуть по любой вкладке отчёта, чтобы открыть её в отдельном окне с возможностью изменения размеров. Закрытие этого отдельного окна вернёт вкладку обратно в Панель управления ПЛИС.

Предупреждения и ошибки, содержащие значок DRC, логически связаны с объектами на схеме. При выборе такой записи соответствующие элементы схемы подсвечиваются красными маркерами. Выбор другой записи DRC перемещает подсветку, а закрытие отсоединённого окна предупреждений или ошибок очищает активную подсветку.
