Меню Файл

Создать  Ctrl-N

Открывает новый проект в новом окне. Изначально проект будет создан как копия текущего выбранного шаблона.

Объединить…  Ctrl-M

Объединяет компоненты другого проекта с текущим проектом. При этом компоненты в объединяемых проектах не должны иметь одинаковых названий.

Открыть…  Ctrl-O

Открывает существующий файл проекта в новом окне.

Открыть недавние

Открывает один из недавно использовавшихся проектов в новом окне без необходимости искать файл через стандартный диалог выбора.

Закрыть  Ctrl-Shift-W

Закрывает все окна, связанные с просматриваемым в данный момент проектом.

Сохранить  Ctrl-S

Сохраняет просматриваемый в данный момент проект, перезаписывая существующий файл проекта.

Сохранить как…  Ctrl-Shift-S

Сохраняет просматриваемый в данный момент проект, предлагая пользователю выбрать имя файла и путь для сохранения.

Экспорт изображения…

Создаёт файлы изображений схем. Диалоговое окно настройки описано в разделе Экспорт изображения.

Экспортировать архив проекта…

Экспортирует весь проект со всеми связанными библиотеками и файлами в единый сжатый архив формата .jar для удобства переноса.

Распаковать и открыть архив проекта…

Позволяет распаковать и открыть ранее созданный архив проекта.

Печать…  Ctrl-P

Отправляет одну или несколько схем на принтер. Настройки печати описаны в разделе Печать.

Настройки…

Открывает окно настроек приложения. (В macOS этот пункт отображается в меню приложения Logisim-evolution.)

Выход  Ctrl-Q

Закрывает все открытые проекты и завершает работу программы. (В macOS этот пункт отображается как Quit в меню приложения Logisim-evolution.)
