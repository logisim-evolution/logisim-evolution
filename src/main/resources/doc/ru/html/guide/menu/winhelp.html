Меню Окно

Свернуть  Ctrl-M

Сворачивает текущее окно в панель задач/док-панель.

Развернуть (Масштабировать в macOS)

Масштабирует текущее окно до его оптимального размера.

Закрыть  Ctrl-W

Закрывает текущее окно.

Показать панель инструментов

Переключает видимость панели инструментов главного окна Logisim-evolution (скрывает или показывает её).

Комбинационный анализ

Открывает или выводит на передний план текущее окно комбинационного анализа без изменения его содержимого.

Настройки

Открывает окно настроек приложения.

названия открытых окон

Выводит выбранное из списка открытых окон окно на передний план.

Меню Справка

Руководство для начинающих

Открывает раздел справки Руководство для начинающих в Руководстве пользователя Logisim.

Руководство пользователя

Открывает главную страницу Руководства пользователя Logisim.

Справка по библиотеке

Открывает раздел справки Справка по встроенным библиотекам компонентов.

Сайт проекта

Открывает в веб-браузере официальный сайт проекта Logisim-evolution.

О программе…

Отображает окно с информацией о версии программы, авторах и лицензии поверх заставки Logisim-evolution. (В macOS этот пункт расположен в меню приложения Logisim-evolution.)

Далее: Руководство пользователя Logisim.
