Меню Моделирование

Моделирование включено  Ctrl-E

Если этот флажок установлен, моделирование схемы работает в реальном времени: значения сигналов на проводниках мгновенно обновляются при изменении элементов схемы или переключении контактов.

Этот режим автоматически отключается, если в схеме обнаруживается бесконечная автоколебательная генерация.

Один шаг моделирования  Ctrl-I

Продвигает моделирование на один шаг распространения сигналов вперёд. Подробнее см. раздел Пошаговый режим.

Для удобства анализа изменённые контакты и концы проводников помечаются синими кружками. Если изменения произошли внутри подсхемы, её графическое обозначение на родительской схеме обводится синей рамкой.

Сбросить моделирование  Ctrl-R

Сбрасывает текущее состояние всех компонентов схемы, возвращая их к исходным значениям, заданным при открытии файла. При просмотре состояния подсхемы сбрасывается вся иерархия проекта.

Моделирование VHDL включено

Эта функция на данный момент не документирована.

Перезапуск моделирования VHDL

Эта функция на данный момент не документирована.

Перейти на уровень выше

При исследовании внутреннего состояния подсхемы этот пункт меню отображает список родительских схем, находящихся выше текущей по иерархии, позволяя быстро вернуться к ним.

Войти в подсхему

Позволяет перейти к исследованию логического состояния конкретного экземпляра подсхемы, расположенного на холсте текущей схемы.

Выполнить полутакт  Ctrl-T

Продвигает тактовый сигнал в моделировании на один шаг (один полутакт) вперёд при пошаговом режиме. Например, тактовый генератор с длительностью высокого/низкого уровней в 1 такт переключит своё состояние на противоположное.

Это удобно при ручной отладке схем с тактовыми импульсами, особенно если тактовый генератор находится во вложенной подсхеме.

Выполнить такт  F2

Выполняет аналогичное действие, но продвигает моделирование на два шага (полный такт). Тактовый генератор с длительностью фаз в 1 такт завершит полный цикл колебания. Другие тактовые генераторы проекта продвигаются пропорционально их параметрам.

Генерация тактов  Ctrl-K

Запускает автоматическую непрерывную генерацию тактовых импульсов. Работает только в том случае, если на схеме присутствуют тактовые генераторы из библиотеки «Монтаж». По умолчанию генерация отключена.

Тактовая частота

Определяет частоту автоматической генерации тактов. Например, частота 8 Гц означает, что тактовые шаги будут происходить 8 раз в секунду.

Пример расчета: Длительность высокого уровня = 2, Длительность низкого уровня = 2. Период тактового генератора составит 2 + 2 = 4 шага моделирования. Частота импульсов генератора будет равна частоте шагов делённой на 4.

При максимальной частоте генератора (длительности фаз 1 и 1) и частоте моделирования 8 Гц, частота импульсов на выходе составит 4 Гц (поскольку полный цикл состоит из такта высокого и такта низкого уровней).

Временная диаграмма…

Открывает панель Временная диаграмма, которая позволяет автоматически фиксировать и отслеживать логические состояния выбранных цепей во времени в процессе моделирования.

Тестовый вектор…

Открывает окно Тестовый вектор, которое позволяет осуществлять автоматическое функциональное тестирование схемы по эталонным таблицам входных и выходных сигналов из тестового файла.

Просмотр кода ассемблера

Открывает окно Просмотр кода ассемблера, отображающее текущее содержимое программного счётчика/регистра адреса и соответствующую инструкцию ассемблера, извлечённую из памяти.
