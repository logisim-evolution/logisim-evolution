Меню ПЛИС

Редактор плат ПЛИС

Открывает Редактор плат ПЛИС (FPGA Board Editor), который предназначен для создания и редактирования XML-файлов описания отладочных плат. Описание платы объединяет её графическое изображение, параметры ПЛИС и тактового генератора, а также прямоугольные области, описывающие переключатели, светодиоды, контакты и другие ресурсы ввода-вывода платы.

Чтобы начать описание платы, щёлкните по пустой области платы и выберите PNG-изображение платы, либо воспользуйтесь кнопкой Загрузить плату для загрузки существующего XML-описания, либо кнопкой Встроенная плата для загрузки одной из плат, поставляемых в комплекте с Logisim-evolution. При загрузке изображения поле названия платы инициализируется из имени файла или названия платы.

Используйте кнопку Настроить ПЛИС для ввода параметров ПЛИС, тактовой частоты и параметров загрузки прошивки. Чтобы определить ресурсы ввода-вывода платы, перетащите прямоугольную область на изображение платы. После создания прямоугольной области Logisim-evolution откроет диалоговое окно для выбора и настройки типа компонента ввода-вывода. Заданные прямоугольные области не должны перекрываться.

Существующие области можно редактировать двойным щёлчком мыши, перемещать перетаскиванием выделенной области и изменять их размер с помощью маркера в правом нижнем углу. Ползунок масштабирования позволяет изменять масштаб отображения при редактировании изображения платы.

Используйте кнопку Сохранить плату для сохранения описания во внешний XML-файл платы. Если в качестве отправной точки была загружена встроенная плата, измените её название перед сохранением во внешний файл, иначе имя внешней платы будет конфликтовать со встроенной.

Панель управления ПЛИС

Открывает Панель управления ПЛИС (FPGA Commander) для проверки правил проектирования (DRC), сопоставления ресурсов схемы с выбранной платой, генерации HDL-кода и файлов ограничений, а также запуска настроенного инструментария синтеза ПЛИС. Глобальные настройки панели управления ПЛИС описаны во вкладке настроек Панели управления ПЛИС.
