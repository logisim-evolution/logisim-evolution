Размещение компонентов в один матрицы

В Logisim-evolution можно разместить на вашей схеме компоненты, расположенные в матрице. Для этого действуйте следующим образом:
Выберите нужный инструмент в навигационной панелиy, например, светодиод, и переместите его на рабочую поверхность, и перед тем, как щёлкнуть по нему, нажмите клавишу X . Тень компонента превратится в маленькую матрицу 2 на 2.

Нажмите на верхнюю левую часть позиции, где вы хотите разместить будущую матрицу. И заполните данные формы. Например: элементы 4x5 с пространством 4.

Если активирована самомаркировка, компоненты будут обозначены как _X_Y, Пример: Led_X1_Y0

Далее: Руководство пользователя.
