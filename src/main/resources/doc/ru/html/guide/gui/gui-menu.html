Меню

Этот раздел быстро объясняет семь меню, которые сопровождают каждое главное окно логистики.

Меню Файл : Это меню позволяет открыть близко для сохранения проектов. Оно дает доступ к предпочтениям и позволяет выйти из программы.

Меню Правка : В основном используется для редактирования, копирования, удаления, дублирования, перемещения элементов чертежа.

Меню проекта : Позволяет управлять библиотеками, изменять порядок в панели навигации, а также изменять внешний вид этой панели. Предоставляет доступ к предпочтениям проекта.

Меню моделирования : Управление симулятором, запуск, остановка, переход в пошаговый режим, запуск часов.

меню FPGA : Редактирование изображения платы FPGA, управление синтезом и загрузкой схемы в плату. .

меню Окно :Минимизируй, расширяй, закрой окна. Перейдите в окно "Комбинированный анализ" (если включено) и настройки приложения.

Меню справки :Перейдите к учебнику, справочнику или ссылке на библиотеки.

Полную информацию о меню вы найдете на страницах, посвященных этой теме.
