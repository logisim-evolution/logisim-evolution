Специальные значения

Формат тестовых векторов поддерживает два специальных значения для более гибкого тестирования:

  • <DC> (Безразлично): При использовании для выходного контакта это значение всегда проходит независимо от фактического выхода. Это полезно, когда вам не важно определённое значение выхода во время теста. Регистр не учитывается (<dc> также работает).
  • <float>: При использовании для входного контакта это подаёт плавающее (высокоомное) значение. При использовании для выходного контакта это ожидает, что выход будет плавающим (неизвестное значение). Регистр не учитывается (<FLOAT> также работает).

Вот пример использования специальных значений:

# Тест с безразличными и плавающими значениями
Input   Enable Output  <set> <seq>
1       1      <DC>    0     0
<float> 0      <float> 0     0

В первом тесте выход помечен как безразличный, поэтому любое выходное значение пройдёт проверку. Во втором тесте вход плавающий, и выход также ожидается плавающим.
