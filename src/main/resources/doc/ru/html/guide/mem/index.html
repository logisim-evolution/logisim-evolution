Компоненты памяти

Компоненты ОЗУ (RAM) и ПЗУ (ROM) — одни из самых востребованных элементов во встроенной библиотеке Logisim-evolution. Однако из-за большого объёма данных, который они способны хранить, они также являются одними из самых сложных в использовании.

Техническую документацию по их работе в схеме можно найти на страницах ОЗУ и ПЗУ в Справке по библиотеке. Данные разделы Руководства пользователя описывают графический интерфейс, позволяющий просматривать и редактировать содержимое памяти.

  • Встроенный шестнадцатеричный редактор Logisim-evolution
  • Редактирование содержимого памяти
  • Контекстное меню и файлы
