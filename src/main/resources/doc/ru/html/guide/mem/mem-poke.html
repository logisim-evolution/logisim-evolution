Редактирование содержимого памяти

Содержимое памяти можно изменять непосредственно на холсте с помощью Инструмента «Нажатие». Однако этот интерфейс имеет ограничения, поэтому для массового или сложного редактирования гораздо удобнее использовать встроенный шестнадцатеричный редактор.

Выбор адреса памяти

Чтобы перейти к другой области памяти, щёлкните левой кнопкой мыши по столбцу адресов (слева от сетки данных). Logisim выделит верхний адрес красной рамкой.

Управление навигацией по адресам:

  • Ввод шестнадцатеричного числа: мгновенно перемещает область просмотра к указанному адресу;
  • Enter или Ctrl + Стрелка вниз — прокручивает память на одну строку вниз;
  • Backspace или Ctrl + Стрелка вверх — прокручивает память на одну строку вверх;
  • Пробел — прокручивает память на одну страницу вниз;
  • Колёсико мыши — плавная вертикальная прокрутка.

Изменение данных

Для изменения конкретного значения щёлкните по нужной ячейке данных внутри матрицы. Logisim выделит эту ячейку красной рамкой.

После этого вы можете ввести новое значение ячейки напрямую с клавиатуры (в шестнадцатеричном формате).

Управление курсором в сетке данных:

  • Пробел или Ctrl + Стрелка вправо — перемещает курсор на одну ячейку вперёд;
  • Backspace или Ctrl + Стрелка влево — возвращает курсор на одну ячейку назад;
  • Enter или Ctrl + Стрелка вниз — перемещает курсор на одну строку вниз;
  • Ctrl + Стрелка вверх (или Колёсико мыши) — перемещает курсор на одну строку вверх.

Чтобы выйти из режима редактирования, просто щёлкните левой кнопкой мыши в любом пустом месте схемы.
