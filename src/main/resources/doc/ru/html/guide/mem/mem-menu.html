Всплывающие меню и файлы

Всплывающие меню

Контекстное меню для компонентов памяти включает в себя на четыре опции больше, чем те, которые являются общими для всех компонентов :

Редактировать содержимое :
вызывает шестнадцатеричный редактор для редактирования содержимого памяти.
Очистить содержимое :
Сбрасывает ПЗУ на 0, а для оперативной памяти на 0 или случайным образом в соответствии с настройками в Опции проекта.
CЗагрузить образ... :
Сбрасывает все значения в памяти к значениям, найденным в файле. Формат файла распознается автоматически, и в противном случае открывается окно для соответствующего выделения.
Сохранить образ... :
Хранит все значения в памяти в файле, используя один из форматов, описанных ниже.

Форматы файлов

  • v2.0 raw сторическая версия Logisim позволяет кодирование повторов.
  • v3.0 hex words plain Близко к версии v2.0, структурирована на словах.
  • v3.0 hex words addressed Как и раньше, с возможностью определения стартового адреса для каждой строки документ.
  • v3.0 hex bytes plain big-endian
    v3.0 hex bytes plain little-endian Организованный в байтовых потоках.
  • v3.0 hex bytes addressed big-endian
    v3.0 hex bytes addressed little-endian Организовано в байтовом потоке, как и раньше, с возможностью определения адреса.
  • Binary data big-endian
    Binary data little-endian Сырой образ памяти и не кодированный в шестнадцатиричном.
  • Ascii byte escape big-endian
    Ascii byte escape little-endian Файл в кодировке ASCII с экранирующим кодом, как в C для непечатаемых символов.
