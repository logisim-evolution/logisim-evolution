HDL IP

В Logisim вы можете использовать VHDL для создания новых компонентов, называемых сущностями VHDL. Эти компоненты затем можно использовать в моделировании, при условии, что в системе установлен и правильно настроен Questa Advanced Simulator.

Важное примечание: библиотека HDL IP не загружается в проект по умолчанию. Чтобы её активировать, выберите в верхнем меню пункт | Проект | > | Загрузить библиотеку | > | Встроенная библиотека | и выберите | HDL IP |. После этого на панели навигатора (сайдбаре) появится папка HDL-IP с доступными инструментами.
