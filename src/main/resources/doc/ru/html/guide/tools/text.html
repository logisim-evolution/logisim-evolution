Инструмент «Текст»

Инструмент «Текст» предназначен исключительно для создания независимых текстовых надписей и аннотаций в произвольных местах холста схемы.

Эти надписи используются только как графические комментарии для человека (например, для указания названия лабораторной работы, вариантов или пояснения логики схемы). Они никак не связаны с компонентами, не влияют на прохождение сигналов и полностью игнорируются при генерации VHDL-кода.

Ограничения редактора

Встроенный текстовый редактор обладает базовым функционалом. Частичное выделение букв мышью внутри созданной надписи или перенос текста на новую строку (многострочный текст в рамках одного блока) не поддерживаются.

Атрибуты

При выборе Инструмента «Текст» в левой панели отображаются его настройки (шрифт, начертание, выравнивание). Эти параметры применяются ко всем новым текстовым надписям, которые вы создаёте последующими щелчками мыши на свободном месте холста.
