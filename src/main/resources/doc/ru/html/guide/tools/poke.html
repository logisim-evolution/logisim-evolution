Инструмент «Нажатие»

Инструмент «Нажатие» предназначен для интерактивного управления текущими состояниями компонентов в процессе моделирования. Его точное действие зависит от выбранного элемента (подробности см. в разделе «Поведение Инструмента Нажатие» в справке к конкретному компоненту).

Инструмент «Нажатие» поддерживается следующими компонентами:

  • Библиотека «Монтаж»: Контакт, Тактовый генератор;
  • Библиотека «Память»: Триггеры D/T/JK/RS, Регистр, Счётчик, Сдвиговый регистр, ОЗУ (RAM), ПЗУ (ROM);
  • Библиотека «Ввод/вывод»: Кнопка, Джойстик, Клавиатура.

Кроме того, щелчок инструментом «Нажатие» по любому проводнику выводит на экран логическое значение, которое передаётся по нему в данный момент (подробнее см. справку по Инструменту разводки).
