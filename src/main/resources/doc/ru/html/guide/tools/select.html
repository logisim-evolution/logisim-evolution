Инструмент «Выбор»

Инструмент «Выбор» предназначен для выбора одного или нескольких компонентов на холсте с целью их последующего перемещения, редактирования или копирования.

Выделение объектов

  • Одиночный клик по компоненту: Снимает текущее выделение и выбирает компонент, по которому был произведён щелчок.
  • Shift + Клик: Инвертирует статус компонента: добавляет его в выделенную группу или исключает из неё.
    Примечание: Это действие не сработает, если комбинация Shift + Клик переназначена на другой инструмент в настройках проекта на вкладке Мышь.
  • Рамка (Лассо): Перетаскивание курсора, начатое на пустом месте холста, создаёт прямоугольную область выделения. Все попавшие в неё компоненты будут выделены (предыдущее выделение сбрасывается).
  • Shift + Рамка: Инвертирует выделение для всех компонентов, попавших в прямоугольник. (Аналогично не сработает при переназначении кнопок мыши).

Перемещение объектов

  • Перетаскивание: Нажатие кнопки мыши на уже выделенном объекте позволяет перемещать всю выделенную группу по холсту.
  • Автосохранение связей: По умолчанию Logisim автоматически перестраивает (иногда укорачивая или удаляя) проводники так, чтобы при перемещении компонентов существующие соединения не разрывались. Удержание клавиши Shift при перетаскивании отключает это поведение.
    Примечание: Вы можете инвертировать это поведение по умолчанию в меню | Проект | > | Параметры… | (на вкладке | Холст | снимите галочку Сохранять соединения при перемещении).
  • Слияние при наложении: Если перетащить выделенную группу поверх других существующих проводников, они автоматически сольются в единый узел и добавятся в выделение. Если после этого вы снова переместите группу, «прилипшие» проводники уедут вместе с ней. Во избежание искажения схем отпускайте кнопку мыши только тогда, когда уверены в правильной позиции.

Особенности копирования и вставки

Стандартные операции вырезания, копирования, вставки и удаления доступны через главное меню Правка.

При дублировании или вставке из буфера обмена Logisim не помещает объекты на холст мгновенно. Вместо этого вставляемая схема отображается в виде полупрозрачных «призраков». Они будут физически добавлены в схему только после того, как вы переместите их на нужное место и снимете выделение. (Такое поведение необходимо, чтобы вставляемые проводники случайно не слились с существующими до того, как вы найдёте для них подходящее свободное место).

Атрибуты

Сам Инструмент «Выбор» не имеет собственных атрибутов. Однако при выборе объектов на панели слева отображаются их атрибуты:

  • Если выбран один компонент, отображаются все его настройки.
  • Если выбрано несколько компонентов, отображаются только общие для них атрибуты. Если их значения совпадают, они выводятся на экран; если различаются — поле остаётся пустым.
  • Проводники игнорируются панелью атрибутов, если в выделение входит хотя бы один другой компонент.

Изменение любого значения в панели атрибутов мгновенно применяется ко всем выбранным компонентам.
