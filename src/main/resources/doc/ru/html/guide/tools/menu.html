Контекстное меню

«Контекстное меню» предназначено для вызова контекстного меню выделенных компонентов на холсте. По умолчанию меню открывается щелчком правой кнопки мыши (или Ctrl + Клик).
Примечание: Назначение кнопок мыши можно переопределить в настройках проекта на вкладке Мышь.

Для большинства компонентов контекстное меню содержит два базовых пункта:

  • Удалить: удаляет выбранный компонент со схемы.
  • Показать атрибуты: выводит свойства компонента в левую панель атрибутов для их просмотра и редактирования.

Некоторые компоненты расширяют базовое меню дополнительными действиями. Например, для подсхем (когда одна схема используется как макрокомпонент внутри другой) добавляется пункт:

  • Просмотреть XXX: открывает внутреннюю логическую структуру подсхемы. При этом значения всех сигналов и состояний отображаются в реальном времени — строго в рамках работы родительской схемы (подробнее см. раздел «Отладка подсхем» Руководства пользователя).

Среди базовых компонентов Logisim собственные расширенные контекстные меню имеют также элементы памяти: ОЗУ (RAM) и ПЗУ (ROM).
