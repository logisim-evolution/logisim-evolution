Окно временной диаграммы

Это окно позволяет просматривать хронограммы (временные диаграммы) ранее выбранных сигналов. Оно содержит меню, панель инструментов, две области отображения и кнопки управления.

Меню | Окна | (Windows) содержит одну опцию, определяющую, должно ли окно всегда оставаться поверх других окон или же оно может уходить на задний план. По умолчанию окно отображается поверх остальных.

Панель инструментов содержит пять кнопок:

  Позволяет запустить или остановить моделирование. Эквивалентно команде меню | Моделирование | → | Моделирование включено |.
  Выполнить один шаг моделирования. Эквивалентно команде меню | Моделирование | → | Один шаг моделирования |.
  Включить или отключить автоматическую генерацию тактовых сигналов sysclk. Эквивалентно команде меню | Моделирование | → | Генерация тактов |.
  Продвинуть системный тактовый сигнал sysclk на половину периода (полутакт). Эквивалентно команде меню | Моделирование | → | Выполнить полутакт |.
  Продвинуть тактовый сигнал clk на период или полупериод. Эквивалентно команде меню | Моделирование | → | Выполнить такт | или | Выполнить полутакт |.

Область отображения справа содержит три столбца: в первом показан значок типа сигнала (одиночный проводник или многоразрядная шина), во втором — имя сигнала, в третьем — значение сигнала под курсором (красная вертикальная линия на графике). Курсор можно перемещать мышью (нажать левую кнопку мыши и тащить). Щелчок правой кнопкой мыши по строке шины открывает контекстное меню с двумя опциями:

  • | Формат | — позволяет выбрать систему счисления для отображения значений сигнала (двоичная, шестнадцатеричная, восьмеричная, знаковая десятичная, беззнаковая десятичная).
  • | Развернуть | — показывает под текущей строкой индивидуальные хронограммы для каждого отдельного бита в составе шины.

Кнопки управления в нижней части окна:

  • Загрузить файл — считывает и отображает результаты измерений из ранее сохранённого текстового файла. В этом режиме интерактивное моделирование недоступно.
  • Экспорт... — экспортирует накопленные данные в текстовый файл с разделителями-табуляциями. Вам будет предложено указать имя файла в окне сохранения. Формат файла прост: первая строка содержит заголовки столбцов (имена сигналов), вторая строка указывает тип тактового сигнала и временной шаг (если выбрано), последующие строки содержат значения сигналов в каждый момент времени. Третья строка показывает начальную инициализацию, а все последующие строки соответствуют каждому полупериоду тактового сигнала sysclk. 
    sysclk     X       a       b       c       
clk:X:1:Hz
0       11      0       0       0       
0       11      0       0       0       
1       01      1       1       0       
0       01      0       1       0       
1       10      1       0       1       
0       10      0       0       1       
1       00      1       1       1       
0       00      0       1       1
  • Экспорт как изображение — сохраняет хронограммы в файл изображения формата PNG. Вам потребуется указать имя файла в окне сохранения.
