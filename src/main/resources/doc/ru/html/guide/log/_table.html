Вкладка «Таблица»

Вкладка «Таблица» отображает накопленные результаты моделирования в табличной форме.

Таблица содержит отдельный столбец для каждого выбранного сигнала. Каждая строка представляет собой снимок состояния схемы после завершения очередного шага моделирования. Повторяющиеся строки (когда значения сигналов не менялись) в таблицу не добавляются. Обратите внимание, что отображаются только последние 400 строк. Некоторые ячейки могут оставаться пустыми, если соответствующий компонент ещё не был добавлен в список отслеживания на момент формирования данной строки.

Отображаемая таблица предназначена только для чтения.
