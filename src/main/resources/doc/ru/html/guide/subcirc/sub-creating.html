Создание схем

Каждый проект Logisim - фактически библиотека схем. В своей простейшей форме каждый проект имеет только одну схему (по умолчанию называемую "main"), но легко можно добавить больше: для этого Вы можете либо нажать на в маленькой панели меню над Панель проводника или или щелкните правой кнопкой мыши на корне дерева в Панель проводника либо выберите | Проект |→| Добавить схему... |, а затем введите любое название для новой схемы, которую вы хотите создать.

Допустим, мы хотим построить мультиплексор 2-в-1 с названием Mux_2To1. После добавления схемы Logisim будет выглядеть следующим образом.

В панели проводника вы можете видеть, что проект теперь содержит две схемы, main и Mux_2To1. Logisim отрисовывает увеличительное стекло поверх значка схемы, рассматриваемой в данный момент; ещё, название этой схемы появляется в заголовке окна.
