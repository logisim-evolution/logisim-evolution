Компаратор с плавающей точкой

Библиотека: Арифметика
Введён в: 3.5
Внешний вид:

Поведение

Этот компонент сравнивает два значения с плавающей запятой (одинарной точности 32 бита или двойной точности 64 бита), поступающие на входы на левой стороне, и устанавливает один из выходов на правой стороне в логическую 1 в зависимости от их соотношения:

  • Выход > (больше) устанавливается в 1, если A > B.
  • Выход = (равно) устанавливается в 1, если A = B.
  • Выход < (меньше) устанавливается в 1, если A < B.

Если хотя бы один из операндов равен специальному значению «не-число» (NaN) (или если на входах присутствует неинициализированное значение (U), высокоимпедансное состояние (Z) или значение ошибки (E), преобразуемые в NaN), то все три выхода сравнения (>, =, <) устанавливаются в 0, а выход ошибки (ERR) устанавливается в логическую 1.

Контакты

Левая сторона, верхний край (вход, разрядность соответствует атрибуту «Разрядность числа с плавающей точкой»)
Входное значение A с плавающей запятой.
Левая сторона, нижний край (вход, разрядность соответствует атрибуту «Разрядность числа с плавающей точкой»)
Входное значение B с плавающей запятой.
Правая сторона, верхний край (выход, разрядность равна 1)
Выход «больше» (>). Принимает значение 1, если A > B, и 0 в противном случае.
Правая сторона, посередине (выход, разрядность равна 1)
Выход «равно» (=). Принимает значение 1, если A = B, и 0 в противном случае.
Правая сторона, нижний край (выход, разрядность равна 1)
Выход «меньше» (<). Принимает значение 1, если A < B, и 0 в противном случае.
Нижняя сторона (выход, разрядность равна 1)
Выход ошибки (ERR). Выдаёт 1, если хотя бы один из операндов является NaN (не-число), и 0 в противном случае.

Атрибуты

Разрядность числа с плавающей точкой
Разрядность (в битах) операндов (32 для одинарной точности или 64 для двойной точности).
