Сумматор

Библиотека: Арифметика
Введён в: 2.0 Beta 11
Внешний вид:

Поведение

Этот компонент складывает два значения, поступающие на левые входы, и выдаёт их сумму на правый выход. Сумматоры можно каскадировать для сложения чисел большей разрядности:

  • Вход переноса (carry-in) — принимает бит от предыдущего каскада;
  • Выход переноса (carry-out) — передаёт сигнал переполнения на следующий сумматор.

Если в слагаемых присутствуют биты с ошибкой (E) или в высокоимпедансном состоянии (Z), выполняется частичное сложение:

  • Корректный результат вычисляется только для младших разрядов — вплоть до первого проблемного бита.
  • Начиная с него и для всех старших разрядов, на выходе будут формироваться значения Z или E соответственно.

Контакты

Левая сторона, верхний край (вход, разрядность соответствует атрибуту «Биты данных»)
Первое слагаемое.
Левая сторона, нижний край (вход, разрядность соответствует атрибуту «Биты данных»)
Второе слагаемое.
Верхняя сторона, отмечен c in (вход, разрядность равна 1)
Значение переноса для прибавления к сумме. Если значение на входе не определено (находится в высокоимпедансном состоянии (Z)), то предполагается, что оно равно 0.
Правая сторона (выход, разрядность соответствует атрибуту «Биты данных»)
Младшие dataBits битов суммы двух входных значений с левой стороны плюс бит переноса cin.
Нижняя сторона, отмечен c out (выход, разрядность равна 1)
Вычисленный бит переноса для суммы. Если сумма входных значений (при интерпретации их как беззнаковых) укладывается в разрядность dataBits, то на этом выходе будет 0, иначе — 1.

Атрибуты

Когда компонент выбран или добавляется на холст, сочетания клавиш от Alt-0 до Alt-9 изменяют значение его атрибута Биты данных.

Биты данных
Разрядность складываемых значений и результата.
