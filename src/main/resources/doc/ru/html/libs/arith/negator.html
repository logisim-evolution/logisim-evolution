Отрицатель

Библиотека: Арифметика
Введён в: 2.0 Beta 22
Внешний вид:

Поведение

Вычисляет отрицание для входного значения в дополнительном коде. Это отрицание осуществляется сохранением всех младших битов до самой младшей 1 (включая её) и инвертированием (дополнением) всех битов старше этого значения.

Если значение для отрицания является минимально возможным отрицательным числом (которое не имеет положительного эквивалента в дополнительном коде заданной разрядности), то результатом отрицания остаётся само это минимальное отрицательное число.

Контакты

Левая сторона (вход, разрядность соответствует атрибуту «Биты данных»)
Значение для отрицания.
Правая сторона, отмечен −x (выход, разрядность соответствует атрибуту «Биты данных»)
Отрицание входного значения. Если входное значение является минимальным отрицательным числом, представимым с помощью dataBits битов, то выходное значение будет равно входному.

Атрибуты

Когда компонент выбран или добавляется на холст, сочетания клавиш от Alt-0 до Alt-9 изменяют значение его атрибута Биты данных.

Биты данных
Разрядность входа и выхода компонента.
