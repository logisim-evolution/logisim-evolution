Умножитель с плавающей точкой

Библиотека: Арифметика
Введён в: 3.5
Внешний вид:

Поведение

Этот компонент умножает два значения с плавающей запятой (одинарной точности 32 бита или двойной точности 64 бита), поступающие на входы на левой стороне, и выдаёт их произведение на выход на правой стороне.

Компонент поддерживает два режима работы, определяемых атрибутом Режим умножения:

  • Умножение — стандартное умножение двух значений: A * B.
  • Совмещённое умножение-сложение (FMA) — выполняет операцию вида A * B + C. При выборе этого режима на верхней стороне компонента появляется дополнительный вход C для слагаемого.

Если в процессе вычисления получается значение «не-число» (NaN), то на выходе ошибки (ERR) устанавливается логическая 1, в противном случае — 0. Если на каком-либо из активных входов присутствует неинициализированное значение (U), высокоимпедансное состояние (Z) или значение ошибки (E), соответствующий операнд интерпретируется как NaN, в результате чего на выходе также формируется значение NaN, а на выходе ошибки устанавливается логическая 1.

Контакты

Левая сторона, верхний край (вход, разрядность соответствует атрибуту «Разрядность числа с плавающей точкой»)
Первый множитель с плавающей запятой.
Левая сторона, нижний край (вход, разрядность соответствует атрибуту «Разрядность числа с плавающей точкой»)
Второй множитель с плавающей запятой.
Верхняя сторона (вход, разрядность соответствует атрибуту «Разрядность числа с плавающей точкой»)
Добавочное слагаемое C (активен только в режиме совмещённого умножения-сложения (FMA)).
Правая сторона (выход, разрядность соответствует атрибуту «Разрядность числа с плавающей точкой»)
Результат умножения (или FMA).
Нижняя сторона (выход, разрядность равна 1)
Выход ошибки (ERR). Выдаёт 1, если результатом операции является NaN (не-число), и 0 в противном случае.

Атрибуты

Разрядность числа с плавающей точкой
Разрядность (в битах) операндов и результата (32 для одинарной точности или 64 для двойной точности).
Режим умножения
Выбор режима работы — Умножение или Совмещённое умножение-сложение (FMA).
