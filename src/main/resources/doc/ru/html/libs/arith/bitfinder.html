Поиск бита

Библиотека: Арифметика
Введён в: 2.6.1
Внешний вид:

Поведение

Компонент принимает многобитное входное значение и находит индекс первого подходящего бита, причём нумерация битов начинается с 0 (младший разряд). Какое именно значение ищется, определяется атрибутом «Тип», как показано в таблице ниже для 8-битного входного значения 11010100:

Тип Выходное значение для 11010100
Младший 1 2
Старший 1 7
Младший 0 0
Старший 0 5

Для поиска младшей 1 результатом будет 2, потому что при нумерации битов с 0 (от младшего к старшему) первая встреченная единица будет во 2-м разряде (разряды 0 и 1 равны 0). Для поиска старшей 1 результатом будет 7, так как самая старшая единица находится в 7-м разряде.

Выход на нижней стороне указывает, был ли вообще найден искомый бит. В приведённых примерах для входа 11010100 нижний выход равен 1 во всех случаях. Но если на вход подать 00000000 и настроить компонент на поиск младшей 1, то нижний выход будет равен 0, а на правом выходе (индекс) также будет 0.

Если в процессе поиска встретится бит, не равный 0 или 1 (то есть находящийся в высокоимпедансном состоянии (Z) или содержащий значение ошибки), то оба выхода примут ошибочное значение. Обратите внимание, что это происходит только в том случае, если проблемный бит встречается до нахождения искомого бита: например, для входного значения x1010100 на выходе всё равно будет 2 при поиске младшей 1, но будут выведены значения ошибки при поиске старшей 1 или старшего 0, поскольку самый старший бит содержит неопределённость (Z или ошибка), а не 0 или 1.

Контакты

Левая сторона (вход, разрядность соответствует атрибуту «Биты данных»)
Многобитный вход, в котором производится поиск бита.
Правая сторона (выход, разрядность вычисляется как описано ниже)
Индекс искомого бита, считая от 0 (младший бит). Разрядность выхода равна минимальному числу битов, необходимому для представления максимального индекса (который на единицу меньше разрядности атрибута «Биты данных»).
Нижняя сторона (выход, разрядность равна 1)
Выход присутствия: равен 1, если искомый бит найден; 0, если все биты входа противоположны искомому; и значению ошибки, если неопределённый или ошибочный бит встретился раньше искомого.

Атрибуты

Когда компонент выбран или добавляется на холст, сочетания клавиш от Alt-0 до Alt-9 изменяют значение его атрибута Биты данных.

Биты данных
Разрядность входа.
Тип
Определяет, какой бит искать (Младший 0, Старший 0, Младший 1 или Старший 1).
