Библиотека HDL-IP

Библиотека HDL-IP содержит компоненты, логику работы которых можно описывать с помощью текстовых форматов описания аппаратуры или логических схем. В библиотеке доступны следующие компоненты:

  • Сущность VHDL — компонент, логика работы которого описывается кодом на языке VHDL.
  • Компонент BLIF — компонент, структура которого задается в текстовом формате BLIF (Berkeley Logic Interchange Format).

Важное примечание: по умолчанию эта библиотека не загружена в проект. Чтобы добавить её, выберите в верхнем меню пункт | Проект | > | Загрузить библиотеку | > | Встроенная библиотека | и выберите библиотеку | HDL IP |. После этого в панели навигатора появится папка HDL-IP.
