Справка по библиотеке

Библиотека Logisim содержит набор специализированных инструментов для интерактивной работы со схемой с помощью щелчков и перетаскиваний мышью на холсте. Большинство из них предназначено для добавления новых элементов в проект, однако ключевые инструменты — такие как Инструмент «Нажатие» и Инструмент «Выбор» — позволяют взаимодействовать с уже размещёнными компонентами и изменять их состояние.

В данном справочном материале приведено подробное описание всех инструментов, входящих в стандартную библиотеку Logisim.
