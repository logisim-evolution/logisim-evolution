Селектор битов

Библиотека: Коммутаторы
Введён в: 2.0.5
Внешний вид:

Поведение

Получая на входе многобитный сигнал, этот компонент разделяет его на несколько групп одинаковой ширины (начиная с младших разрядов входа) и передает на выход ту группу, индекс которой задан на выбирающем входе.

Например, если на вход поступает 8-битное значение 01010101, а разрядность выхода задана равной 3, то группа 0 будет состоять из трех младших битов 101, группа 1 — из следующих трех битов 010, а группа 2 — из оставшихся двух битов, дополненных ведущим нулем до трех: 001. (Все биты за пределами старшего разряда входа при необходимости заполняются нулями.) Выбирающий вход принимает индекс выбираемой группы; например, если на выбирающем входе задано значение 3 (которому нет соответствия в исходных данных), то на выходе установится значение 000.

Контакты (при направлении компонента вправо):

Левый контакт (вход, разрядность соответствует атрибуту Биты данных):
Входные данные, из которых выбирается группа битов для вывода.
Правый контакт (выход, разрядность соответствует атрибуту Выходные биты):
Группа битов из входных данных, выбранная по значению на выбирающем входе.
Нижний контакт (вход, разрядность равна количеству групп, определяемому как отношение Биты данных к Выходные биты с округлением вверх):
Вход выбора: определяет индекс группы битов, которая передается на выход.

Атрибуты

Когда компонент выбран или добавляется, цифровые клавиши от 0 до 9 меняют его атрибут Выходные биты, комбинации клавиш от Alt-0 до Alt-9 меняют его атрибут Биты данных, а клавиши со стрелками — его атрибут Направление.

Направление
Направление компонента (его выхода относительно его входа).
Биты данных
Разрядность входа данных компонента.
Выходные биты
Разрядность выхода компонента (ширина одной выбираемой группы битов).
