Демультиплексор

Библиотека: Коммутаторы
Введён в: 2.0 Beta 11
Внешний вид:

Поведение

Копирует значение со входа на левой стороне строго на один из выходов на правой стороне. Выбор конкретного выхода, на который транслируется входной сигнал, определяется текущим значением на управляющем входе.

Контакты (при направлении компонента вправо, при расположении управляющего входа снизу/слева):

Левый контакт (вход, разрядность соответствует атрибуту Биты данных):
Входное значение, которое будет перенаправлено на один из выходов на правой стороне.
Контакты на правой стороне, переменное количество (выходы, разрядность соответствует атрибуту Биты данных):
Информационные выходы. Они нумеруются по порядку сверху вниз, начиная с 0. Выбранный выход (индекс которого совпадает с текущим значением управляющего входа) повторяет значение на левом входе. Все остальные выходы принимают уровень логического 0 или переводятся в высокоимпедансное состояние (Z) (в зависимости от значения атрибута Три состояния). Если управляющий вход содержит неопределённые биты, то значения на всех выходах будут неинициализированными (U).
Нижний контакт, слева (вход, разрядность равна 1):
Вход разрешения (EN). При логическом 0 на этом входе все выходы демультиплексора переводятся в отключенное состояние (зависит от атрибута При отключенном выходе), независимо от значений на информационном и управляющем входах.
Нижний контакт, справа, отмечен серым кружком (вход, разрядность соответствует атрибуту Разрядность адреса):
Управляющий (выбирающий) вход. Значение на этом входе определяет, на какой из выходов на правой стороне будут скоммутированы данные с левого входа.

Атрибуты

Когда компонент выбран или добавляется, цифровые клавиши от 1 до 4 меняют его атрибут Разрядность адреса, комбинации клавиш от Alt-0 до Alt-9 меняют его атрибут Биты данных, а клавиши со стрелками — его атрибут Направление.

Направление
Направление компонента (определяющее сторону расположения информационных выходов).
Управляющий вход
Расположение управляющего входа и входа разрешения относительно компонента.
Разрядность адреса
Разрядность управляющего (выбирающего) входа компонента. Количество информационных выходов демультиплексора равно 2Разрядность_адреса.
Биты данных
Разрядность коммутируемых информационных данных.
Три состояния
Определяет состояние невыбранных выходов демультиплексора: третье состояние (Z) (Да) или логический ноль (Нет).
При отключенном выходе
Определяет состояние выходов при отключении компонента (когда на входе разрешения логический 0). Доступны варианты: Z-состояние (высокоимпедансное состояние (Z) на выходе) или Ноль.
Вход разрешения (EN)
Определяет наличие у компонента входа разрешения. Этот параметр оставлен в основном для совместимости со схемами, созданными в старых версиях Logisim, где вход разрешения отсутствовал.
