Буфер

Библиотека: Элементы
Введён в: 2.0 Beta 1
Внешний вид:
ANSI: IEC:

Поведение

Буфер просто передает на свой выход (справа) любое входное значение, полученное на входе (слева). Способ представления данных на выходе определяется атрибутом Выходное значение. Ниже приведена таблица истинности для одноразрядного буфера.

 Вход   Выход 
  0/1   0/плав. плав./1
0 0 0 U
1 1 U 1
U U U U
E E E E

Если на входе присутствует неинициализированное значение (U), то на выходе также будет неинициализированное значение. Если на входе присутствует значение ошибки (E), то на выходе также будет ошибка.

Буферы — наименее полезные из логических элементов, представленных в Logisim; их присутствие в библиотеке «Элементы» обусловлено скорее стремлением к полноте (наличие компонента для каждой возможной таблицы истинности с одним входом), нежели практической необходимостью. Тем не менее, они могут быть полезны для того, чтобы гарантировать передачу сигналов по проводнику только в одном направлении.

Контакты (предполагается, что компонент направлен вправо)

Левая сторона (вход):
Вход компонента. Разрядность соответствует атрибуту Биты данных.
Правая сторона (выход):
Выход, значение которого совпадает со входным значением. Разрядность соответствует атрибуту Биты данных. Режим представления данных определяется атрибутом Выходное значение.

Атрибуты

Когда компонент выбран или добавляется, комбинации клавиш от Alt-0 до Alt-9 изменяют его атрибут Биты данных, а клавиши со стрелками изменяют его направление (Направление).

Направление
Направление компонента (его выхода относительно входа).
Биты данных
Разрядность входа и выхода компонента.
Выходное значение
Указывает, как логические 0 и 1 преобразуются в выходные значения. По умолчанию ложный результат (0) передается низким уровнем напряжения (0), а истинный (1) — высоким уровнем напряжения (1), но одно из этих значений может быть заменено на высокоимпедансное состояние (Z). Это позволяет организовывать проводные соединения ИЛИ/И, как описано в документации на элементы И/ИЛИ/И-НЕ/ИЛИ-НЕ.
Метка
Текст метки, связанной с элементом.
Шрифт метки
Шрифт для отображения текста метки.

Поведение инструмента «Нажатие»

Нет.
