Расширитель разрядности

Библиотека: Монтаж
Введён в: 2.5.0 (в библиотеке Базовые, перемещён в библиотеку Монтаж в 2.7.0)
Внешний вид:

Поведение

Расширитель разрядности изменяет разрядность (битовую ширину) входного значения. При уменьшении разрядности входное значение усекается до младших битов. При увеличении разрядности младшие биты сохраняются, а для дополнительных старших битов можно выбрать один из режимов заполнения: заполнение нулями, заполнение единицами, знаковое расширение (дублирование старшего бита входа) или заполнение внешним сигналом через дополнительный однобитный вход.

Контакты

Левая сторона (вход, разрядность задаётся атрибутом Разрядность входа)

Многобитный вход для преобразуемого значения.

Правая сторона (выход, разрядность задаётся атрибутом Разрядность выхода)

Выход с результатом преобразования.

Верхняя сторона (вход, разрядность — 1)

Задаёт значение дополнительных старших битов на выходе. Этот контакт доступен только в том случае, если атрибут Тип расширения имеет значение Вход.

Атрибуты

Когда компонент выбран или добавляется, клавиши от 0 до 9 меняют его атрибут Разрядность входа, а комбинации от Alt-0 до Alt-9 меняют атрибут Разрядность выхода.

Разрядность входа
Разрядность входного значения.
Разрядность выхода
Разрядность выходного значения.
Тип расширения
Если разрядность выхода больше разрядности входа, этот атрибут определяет способ заполнения дополнительных старших битов на выходе. При значении Ноль или Единица дополнительные биты заполняются нулями или единицами соответственно. При значении Знак дополнительные биты заполняются копией знакового (самого старшего) бита входного значения. При значении Вход на верхней стороне компонента появляется дополнительный однобитный вход, значение на котором используется для заполнения дополнительных битов.
