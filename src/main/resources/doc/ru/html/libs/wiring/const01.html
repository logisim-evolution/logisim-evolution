Питание/Земля

Библиотека: Монтаж
Введён в: 2.7.0
Внешний вид:

Поведение

Компонент выдаёт на проводник постоянное логическое значение. Компонент Питание (), обозначаемый треугольником, всегда выдаёт логическую 1 (или, если атрибут Биты данных больше 1, — значение из одних единиц). Компонент Земля (), обозначаемый символом из трёх укорачивающихся горизонтальных линий, всегда выдаёт логический 0 (или, если атрибут Биты данных больше 1, — значение из одних нулей).

Ту же функцию выполняет более универсальный компонент Константа. Компоненты Питание и Земля предпочтительны лишь потому, что являются общепринятыми символами в электронике.

Контакты

Каждый компонент имеет единственный контакт — выход, разрядность которого определяется атрибутом Биты данных. На этом выходе всегда присутствует одно и то же значение: у компонента Земля — из одних нулей, у компонента Питание — из одних единиц.

Атрибуты

Когда компонент выбран или добавляется, комбинации от Alt-0 до Alt-9 меняют его атрибут Биты данных (клавиши можно нажимать последовательно: например, Alt-1 Alt-2 задаёт значение 12). Клавиши со стрелками меняют атрибут Направление.

Направление
Направление, в котором обращено «острие» стрелки компонента; противоположно расположению контакта.
Биты данных
Разрядность значения, выдаваемого на проводник.

Поведение Инструмента «Нажатие»

Не предусмотрено.
