Шестнадцатеричный индикатор

Библиотека: Ввод/вывод
Введён в: 2.2.0
Внешний вид:

Поведение

Отображает шестнадцатеричную цифру, соответствующую значению на четырёхбитном входе, используя встроенную семисегментную структуру. Если на каком-либо бите входа присутствует нелогическое значение (например, высокоимпедансное состояние Z или ошибка E), то индикатор не горит (все сегменты выключены). Отображением десятичной точки управляет отдельный однобитный вход.

Контакты

Нижняя сторона, первый слева (вход, разрядность 4)
Этот вход интерпретируется как беззнаковое четырёхбитное число, соответствующая шестнадцатеричная цифра которого отображается на индикаторе. Если какой-либо бит входа имеет высокоимпедансное состояние Z или состояние ошибки E, индикатор гаснет.
Нижняя сторона, второй слева (вход, разрядность 1)
Управляет десятичной точкой. Если этот вход не подключен, десятичная точка остаётся выключенной.

Атрибуты

Цвет (вкл.)
Цвет, используемый для отображения сегментов и десятичной точки, когда они включены (по умолчанию красный).
Цвет (выкл.)
Цвет, используемый для отображения сегментов и десятичной точки, когда они выключены (по умолчанию серый).
Цвет фона
Цвет, которым отрисовывается фон индикатора (по умолчанию прозрачный).
Есть десятичная точка
Если значение равно «да», то десятичная точка доступна, если «нет» — она скрыта.
Метка
Текстовая метка, связанная с компонентом.
Положение метки
Положение метки относительно компонента.
Шрифт метки
Шрифт, используемый для отрисовки метки.
Отображать метку
Определяет, отображается ли метка.

Поведение Инструмента «Нажатие»

Нет.
