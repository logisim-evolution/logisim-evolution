Библиотека Ввод/вывод

Библиотека Ввод/вывод включает в себя компоненты, предназначенные для взаимодействия пользователя со схемой (кнопки, клавиатуры, выключатели) и отображения результатов её работы (светодиоды, индикаторы, терминалы, видео).

Компоненты

  Кнопка
  DIP-переключатель
  Джойстик
  Клавиатура
  Светодиод
  RGB-светодиод
  7-сегментный индикатор
  Шестнадцатеричный индикатор
  Светодиодная матрица
  Терминал
  Telnet
  Порт ввода-вывода
  Локальная шина Reptar
  RGB видео
