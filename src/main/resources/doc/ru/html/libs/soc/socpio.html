Параллельный расширитель ввода-вывода SoC (PIO)

Библиотека: Компоненты SoC
Появился в версии: 3.2

Параллельный расширитель ввода-вывода (PIO) предоставляет линии цифрового ввода-вывода общего назначения для процессора в системе SoC.

Подробное описание работы и интерфейса расширителя находится на стадии разработки.
