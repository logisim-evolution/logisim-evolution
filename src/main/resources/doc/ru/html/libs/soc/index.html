Библиотека SoC (System on Chip)

Библиотека компонентов систем на кристалле (SoC) содержит полный набор элементов, необходимых для построения и функционального моделирования микропроцессорных систем.

  • Минимальные требования для моделирования
  • Встроенный ассемблер
  • Встроенный дизассемблер
  Имитационная модель процессора RISC-V (расширение IM)
  Имитационная модель процессора Nios2s
  Имитационная модель шины SoC
  Имитационная модель памяти
  Параллельный расширитель ввода-вывода
  Экран VGA
  Контроллер DMA
  Компонент JTAG UART
