Имитационная модель процессора RISC-V (расширение IM)

Библиотека: Компоненты SoC
Появился в версии: 3.2

Имитационная модель процессора RISC-V выполняет программы на ассемблере RISC-V с поддержкой базового набора инструкций Integer (I) и расширения Multiply (M).

Подробное описание работы и интерфейса этой модели находится на стадии разработки.
