Configurer Questa Advanced Simulator

Questa Advanced Simulator peut être utilisé en option pour valider des entités VHDL et est requis pour toute simulation utilisant une entité VHDL.

Par défaut, la validation VHDL avec Questa Advanced Simulator est désactivée. Pour l'activer, vous devez vous rendre dans le menu Fenêtre , Références , puis dans l'onglet Logiciels tiers . Vous pouvez ensuite activer la validation en cliquant sur la case à cocher Utiliser Questa Advanced Simulator pour valider les entités VHDL .

Juste en dessous de cette option, vous pouvez également définir le chemin d'accès au répertoire d'installation de Questa Advanced Simulator. Pour ce faire, cliquez simplement sur le bouton Parcourir ... et sélectionnez le répertoire correspondant. Si vous activez la validation, mais ne définissez pas le chemin, vous serez invité à le sélectionner la prochaine fois qu'une validation VHDL aura lieu.

Note:Vous serez également invité à entrer le chemin si vous essayez d'exécuter une simulation d'un circuit qui inclut une entité VHDL.
