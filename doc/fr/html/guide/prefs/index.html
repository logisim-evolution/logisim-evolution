Préférences d'application

Logisim prend en charge deux catégories d'options de configuration: préférences d'application et options de projet. Les préférences de l'application traitent des préférences qui couvrent tous les projets ouverts, tandis que les options de projet sont spécifiques à ce projet. Cette section traite des préférences d'application. Les options de projet sont décrites dans une autre section.

Vous pouvez afficher et modifier les préférences de l'application via le menu | Fichier |→| Préférences ...  | (ou, sous Mac OS, le menu Logisim), une fenêtre apparaîtra avec plusieurs onglets. Nous discuterons de ces onglets séparément, puis nous verrons comment les préférences peuvent être configurées à partir de la ligne de commande.
