Le menu Fichier

Nouveau  Ctrl-N

Ouvre un nouveau projet dans une nouvelle fenêtre. Le projet sera initialement une copie de la sélection dans application préférence / Modèle.

Fusionner...  Ctrl-M

Fusionne les composant d'un projet avec le projet courant, les composants des deux projets ne doivent pas avoir de nom similaire.

Ouvrir...  Ctrl-O

Ouvre un fichier existant en tant que projet dans une nouvelle fenêtre.

Ouvrir récent

Ouvre un projet récemment ouvert dans une nouvelle fenêtre sans boîte de dialogue de sélection de fichiers.

Femer  Ctrl-Maj-W

Ferme toutes les fenêtres associées au projet actuellement affiché. Une demande de confirmation sera affichée si le projet n'est pas déjà sauvegardé.

Sauvegarder  Ctrl-S

Enregistre le projet actuellement affiché, en écrasant ce qui était précédemment dans le fichier.

Sauvegarder comme...  Ctrl-Maj-S

Enregistre le projet actuellement affiché, en invitant l'utilisateur à préciser un nouveau nom de fichier.

Exporter Image...

Crée des fichiers images correspondant aux circuits. La boîte de dialogue de configuration est décrite dans exportation d'images.

Imprimer...  Ctrl-P

Envoie un ou des circuits à une imprimante. La boîte de dialogue de configuration est décrite dans imprimer.

Préférences...

Affiche le dialogue préférence d' application . (Sur les systèmes Mac OS, cela apparaîtra dans le menu Logisim.).

Sortir  Ctrl-Q

Ferme tous les projets actuellement ouverts et termine Logisim. (Sur les systèmes Mac OS, cela apparaîtra comme Quitter dans le menu Logisim.).
