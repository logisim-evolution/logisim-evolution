Le menu projet

Ajouter Circuit

Ajoute un nouveau circuit dans le projet en cours. Logisim insistera pour que vous spécifiiez un nom pour le nouveau circuit. Le nom ne doit correspondre à aucun circuit existant dans le projet.

Charger une librairie

Charge une librairie dans le projet. Vous pouvez charger trois types de librairies, comme expliqué ailleurs dans le Guide de l'utilisateur..

Enlever les libraires

Décharge les librairies actuelles du projet. Logisim ne vous permet pas de décharger les librairies utilisées par le projet, y compris les librairies contenant des composants apparaissant dans les circuits de projet, ainsi que celles contenant des outils qui apparaissent dans la barre d'outils ou qui sont mappées à la souris.

Monter le Circuit

Déplace le circuit actuellement sélectionné d'un d'une ligne vers le haut dans la liste des circuits du projet visible dans le volet de l'explorateur.

Descendre le Circuit

Déplace le circuit actuellement affiché d'une ligne vers le bas dans la liste des circuits du projet visible dans le volet de l'explorateur.

Définir comme circuit principal

Définis le circuit actuellement affiché comme étant le circuit principal du projet. Cet élément de menu sera grisé si le circuit actuel est déjà le circuit principal du projet. Circuit principal: signifie simplement qu'il sera affiché en premier lorsque l'on ouvre le projet.

Supprimer le circuit

Supprime le circuit actuellement affiché du projet. Logisim vous empêchera de supprimer des circuits utilisés comme sous-circuits et vous empêchera de supprimer le dernier circuit d'un projet.

Revenir à l'apparence par défaut

Si vous avez modifié l'apparence du circuit, cet élément de menu ramène l'apparence par défaut. L'élément de menu est activé uniquement lors de l'édition de l'apparence du circuit. L'aspect de l'apparence par défaut peut être modifié par la propriété Nouvelle apparence de la librairie.

Afficher la barre d'outils

Modifie le volet de l'explorateur pour afficher une liste des circuits du projet et des bibliothèques chargées.

Afficher l'arborescence de simulation

Modifie le volet de l'explorateur pour afficher la hiérarchie des sous-circuits dans la simulation en cours.

Editer le dessin du Circuit

Modifie l'espace de travail pour qu'il affiche l'édition du schéma du circuit. Cet élément de menu est activé quand l'espace de travail affiche l'éditeur d'apparence.

Editer l'apparence du circuit

Permets de modifier la représentation du circuit lorsqu'il est utilisé comme sous-circuit dans un autre circuit. Cette option de menu vous permet de dessiner une apparence différente pour le sous-circuit. Modifie l'espace de travail pour qu'il affiche l'éditeur d'apparence.

Analyse combinatoire

Note: Ce menu peut être désactivé dans certaines versions de logisim-evolution. Il peut être réactivé par des options de la ligne de commande.

Calcule une table de vérité et des expressions booléennes correspondant au circuit en cours, en les affichant dans la fenêtre d'analyse combinatoire. Le processus d'analyse ne sera valable que pour les circuits combinatoires. Une description complète du processus d’analyse est décrite dans la section Analyse combinatoire.

Statistique du circuit

Affiche une boîte de dialogue contenant des statistiques sur les composants utilisés par le circuit actuellement affiché. Le dialogue comprend un tableau à cinq colonnes:

  • Composant: le nom du composant.
  • Librairie: Nom de la Librairie à partir de laquelle le composant est présenté.
  • Simple: nombre de fois où ce composant apparaît directement dans le circuit visualisé.
  • Unique: nombre de fois où ce composant apparaît dans la hiérarchie du circuit, où chaque sous-circuit de la hiérarchie est compté une seule fois.
  • Récursif: nombre de fois où ce composant apparaît dans la hiérarchie du circuit, où nous comptons chaque sous-circuit autant de fois qu'il apparaît dans la hiérarchie.

La distinction entre "Unique" et "Recursive" est plus facile à expliquer en considérant le multiplexeur 4:1 construit avec trois multiplexeurs 2:1 comme dans la section Utiliser des sous-circuits . Le multiplexeur 2:1 contient deux portes ET (et le circuit 4:1 n'en contient aucune), de sorte que le nombre "Unique" dénombrant les portes ET serait 2; mais si vous deviez construire le multiplexeur 4:1 en utilisant ce diagramme, vous auriez réellement besoin de 2 portes ET pour chacun des trois multiplexeurs 2:1, donc le nombre "récursif" est 6.

Si vous utilisez des circuits d'une bibliothèque Logisim chargée, ces composants sont considérés comme des "boîtes noires": le contenu des circuits de la bibliothèque n'est pas inclus dans les comptes uniques et récursifs.
