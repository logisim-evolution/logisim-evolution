Fenêtre Chronogramme

Cette fenêtre permet la visualisation des signaux sélectionnés précédemment. Vous remarquerez le menu, la barre d'outils, deux zones d'affichage et les boutons.

Le menu windows ne contient qu'une option pour préciser si la fenêtre reste affichée au-dessus des autres ou si elle peut passer en arrière-plan. Par défaut elle reste au-dessus.

la barre de menu dispose de cinq boutons

  Permets d'activer la simulation ou de la stopper. Équivalente du menu | Simulation | | Simulation enclenchée |
  Permets d'avancer d'un tic dans la simulation. Équivalente du menu | Simulation | | un seul tic |
  Active ou désactive l'horloge sysclk. Équivalente du menu | Simulation | | tics activés |
  Avance d'une demi-période l'horloge sysclk. Équivalente du menu | Simulation | | 'Un seul tic |
Avance d'une période ou d'une demi-période l'horloge clk. Équivalente du menu | Simulation | | 'Clk' un pas |

La zone d'affichage Sur la droite trois colonnes: la première présente une icône du type de signal (bus ou simple) puis nous avons le nom et enfin la valeur du signal sous le curseur (la ligne rouge dans le graphe). Il est possible de déplacer le curseur à l'aide de la souris, il suffit de cliquer et déplacer. Un clic-droit sur la ligne d'un signal de type bus affiche un menu contextuel avec deux options:

  • | Format |: Ce menu ouvre une sélection de la base d'affichage des signaux (binaire,hexadécimal,octal,signer,non signer)
  • | Exploser |: Affiche, au-dessous, le chronogramme de chaque signal du bus.

Les boutons: On observe trois boutons au fond de notre fenêtre.

  • Chager fichier Charge et affiche le résultat depuis un fichier préalablement sauvegardé. Cela ne permet pas de paramétrer la simulation.
  • Exporter... Exporte les données dans un fichier text. Vous devrez préciser le nom du fichier dans la fenêtre de sauvegarde. Le format du fichier est simple. La première ligne contient le nom des colonnes séparées par une tabulation. La seconde ligne présente le signal et l'échelle de temps si elle a été sélectionnée. Les lignes suivantes présente la valeur ses signaux séparés par une tabulation. La troisième ligne est l'initialisation puis chaque ligne représente un demi-cyle de l'horloge sysclk.
