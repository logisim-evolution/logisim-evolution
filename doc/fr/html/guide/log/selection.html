Sélection des signaux

Cette fenêtre vous permet de sélectionner les valeurs à inclure dans le journal. La fenêtre ci-dessous correspond au circuit suivant :

L'onglet est divisé en trois zones verticales. Le premier (le plus à gauche) est une liste de tous les composants du circuit dont les valeurs peuvent être consignées. Parmi les bibliothèques intégrées, les types de composants suivants prennent en charge la journalisation

Librairie câblage : les broches,sonde et les horloges.
Librairie Entrée/Sortie : les boutons,led.
Librairie Mémoire : Tous les composants à l'exception des ROM.
La valeur des étiquettes sera présentée si celle-ci est associée a un composant dans les autres cas c'est le nom et l'emplacement dans le circuit que sera utilisé pour référence.
Tous les sous-circuits apparaîtront également dans la liste, un double-clic sur l'élément permet la sélection des points de celui-ci. Un composant RAM se comporte un peu de la même manière et les 256 premières adresses seront présentées et sélectionnables. Seuls les composants éligibles peuvent être sélectionnés. Le bouton ajouter>> sera activé seulement pour les composants prévus pour l'insertion.

La dernière zone verticale (la plus à droite) répertorie les composants sélectionnés. De plus, il indique la base dans laquelle les valeurs multibits du composant seront enregistrées. L'horloge sysclk doit y figurer, elle ne sera pas affichée dans le graphe.

La zone au centre contient une liste de boutons.

  • Ajouter >> : Ajoute les éléments actuellement sélectionnés du côté gauche dans la sélection. Il est actif seulement pour les composants éligibles.
  • Change la base : Permet la sélection successive d'une des ces bases : 2 (binaire), 10 (décimal) et 16 (hexadécimal).
  • Monter :déplace le composant actuellement sélectionné vers le haut de la liste.
  • Descendre : Déplace le composant actuellement sélectionné vers le bas de la liste.
  • << Enlever : supprime le composant actuellement sélectionné.

La quatrième partie au-dessous permet de définir si l'on désire un affichage du temps au-dessus du graphe. Pour cela il est nécessaire de cocher la case "Référence temporelle" et sélectionner le signal qui sera la référence. La fréquence définira l'échelle de temps affichée. Cela ne règle absolument pas la vitesse d'exécution de la simulation du circuit. Par exemple ci-dessous nous avons sélectionné le signal B0 et une fréquence de 100hz.
