Editer la table de vérité

En ouvrant la fenêtre d'analyse combinatoire, vous verrez cinq onglets.

Ce document vous apporte de l'aide pour les trois premiers onglets: Entrées, Sorties et Table. Dans la suite du guide, vous trouverez une description des deux onglets: Expression et Karnaugh.

Les onglets entrées et sorties

L'onglet Entrées vous permet d'afficher et éditer la liste des entrées. Pour ajouter des entrées, il suffit de les saisir dans le champ prévu en bas et de cliquer sur Ajouter. Si vous voulez renommer une entrée existante, sélectionnez-la dans la partie du haut à gauche, tapez le nouveau nom et cliquez sur Renommer.

Pour supprimer une entrée, il suffit de la sélectionner dans la liste et cliquer sur Enlever Vous pouvez aussi réorganiser les entrées (ce qui impact sur l'ordre des colonnes dans la table de vérité et la génération du circuit) en utilisant les boutons monter ou descendre sur une entrée.

Toutes les actions impactent directement la présentation de la table de vérité.

L'onglet Sorties fonctionne exactement de la même manière que l'onglet des entrées, sauf qu'il s'agit de la liste des sorties.

L'onglet Table

Le seul élément dans l'onglet Table représente la table de vérité courante, elle est organisée dans un ordre conventionnel, avec les entrées en colonnes à gauche et les sorties en colonnes à droite.

Vous pouvez éditer les valeurs qui apparaissent dans les colonnes de sortie en cliquant sur la valeur choisie. Les valeurs changent de manière cyclique de ,1, et _ (représentant une valeur sans indéterminée). Comme nous le verrons à la page suivante, une valeur sans indéterminée permet l'élaboration des expressions minimales avec une certaine flexibilité.

Vous pouvez également naviguer et éditer la table de vérité avec le clavier. Et vous pouvez copier-coller des valeurs en utilisant le presse-papier. Le presse-papier peut être transféré vers n'importe quelle application qui supporte du texte avec des séparateurs de tabulation (comme un tableur).

Si la table de vérité est basée sur un circuit existant, vous pourriez voir des carrés roses dans les colonnes des sorties avec des "!!" à l'intérieur. Ceci correspond à des erreurs durant le calcul des valeurs pour cette ligne. Soit le circuit semblait osciller, ou la sortie était une erreur elle-même (ce qui devrait être représenté par un câble rouge dans le circuit Logisim). En passant la souris sur l'entrée, vous verrez apparaître une astuce avec une description du type d'erreur. Si vous cliquez sur une entrée erronée, vous vous retrouverez dans le cycle 0-1-_; il n'y a pas moyen de revenir en arrière.
