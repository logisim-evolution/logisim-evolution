La table des attributs

La plupart des composants ont des attributs, qui sont des propriétés pour configurer l'apparence et le comportement des composants. La table des attributs permet d'afficher les valeurs des attributs.

Pour sélectionner les attributs d'un composant, cliquez sur le composant en utilisant l'outil d'édition(). Il est aussi possible de faire un clic-droit (ou contrôle-clic) sur le composant et choisir Afficher les attributs du menu contextuel ou encore en utilisant l'outil pousser () ou l'outil texte () .

L'écran ci-dessous illustre ce qui se passe après avoir sélectionné l'entrée du haut du circuit XOR et de défiler la table vers le bas pour voir le champ attribut Police du Label et sa valeur.

Pour modifier la valeur d'un attribut, cliquez sur cette valeur. L'interface de modification dépendra du type attribut à modifier. Dans le cas de la Police du Label, une boîte de dialogue apparaîtra pour sélectionner la nouvelle police; mais dans le cas de certains attributs (comme le label), il vous sera proposé un champ texte, alors que dans d'autres cas, un menu déroulant vous permettra de sélectionner une valeur.

Chaque type de composant a un ensemble différent d'attributs; pour apprendre ce qu'ils signifient, le document approprié se trouve dans la Librairie de références.

Si vous avez sélectionné plusieurs composants en utilisant l'outil d'édition, alors la table d'attribut ne présentera que les attributs communs aux différents composants sélectionnés (à l'exclusion des câbles). Si les composants sélectionnés n'ont pas la même valeur pour un attribut, alors la valeur pour cet attribut sera un espace blanc. Vous pouvez changer la valeur pour tous les attributs sélectionnés en une fois en l'éditant.
