Компоненты памяти

Компоненты ОЗУ и ПЗУ - два из наиболее полезных компонентов во встроенной библиотеке Logisim. Однако, в силу большого объёма информации, которую они могут хранить, они также - два из наиболее сложных компонентов.

Справка по библиотеке Документацию по их работе в схеме можно найти на страницах ОЗУ и ПЗУ Справка по библиотеке. Эти разделы Руководство пользователя объясняют интерфейс, позволяющий пользователю просматривать и изменять содержимое памяти.
