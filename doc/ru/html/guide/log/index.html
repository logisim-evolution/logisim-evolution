Выбор сигналов
Окно хронограммы

При тестировании больших схем и при документировании поведения схемы, может быть полезен журнал поведения схемы. Это задача для модуля ведения журнала Logisim, который позволяет вам выбрать компоненты, чьи значения должны быть записаны в журнал; по желанию, вы можете указать файл, в который будет помещён журнал.

Вы можете войти в модуль ведения журнала через пункт | Моделировать |→| Запись в журнал... |. Это вызовет окно Выбор сигналов.

Нижеприведенная схема является иллюстративным примером модуля синхронизации.

Это часы, которые служат драйвером для отображения сигналов. Симуляция знает два отдельных часы. Один является обязательным и называется sysclk, а другой необязательным и вторичным именем clk.

Примечание. Необходимо, чтобы в вашей схеме отображались часы с именем sysclk. Он будет использоваться в качестве временной базы для модуля синхронизации. Не существенен, чтобы он был подключен к вашей цепи. Это в принципе самый быстрый и установлен на рабочий цикл 1/1 tic.
