Beim Testen einer größeren Schaltung und zur Dokumentation des Verhaltens einer Schaltung, gibt es die Möglichkeit, das Verhalten der Schaltung zu protokollieren. Dies geschieht durch das Log-Modul von Logisim, das es Ihnen erlaubt, die Bauelemente auszuwählen, deren Werte protokolliert werden sollen. Darüber hinaus können Sie auswählen, in welche Datei diese Daten geschrieben werden sollen.

Sie erreichen das Log-Modul über den menü | Simulieren |→| Loggen... |. Dies wird ein Fenster mit drei Registerkarten öffnen.

These are the clocks that serve as a driver for displaying signals. The simulation knows two particular clocks. One is imperative and named sysclk and another optional and secondary named clk

Note: it is imperative that a clock named sysclk appears in your circuit. It will be used as a time base by the chronogram module. It does not have to be connected to your circuit. It is in principle the fastest and is set to a duty cycle 1/1 tic.
